## Start

Triggers are used to determine where Command are executed.

## Enter Scene Event

The place where the command is executed when the player enters the scenario map.

![](img/Understand/005/001.png=600-)

## Object Events

Scene objects have multiple trigger events

![](img/Understand/005/002.png=600-)

<table>
<tr><td>Click event</td><td>Event page executed when left mouse button or space is pressed against the scene object</td></tr>
<tr><td>Touch events</td><td>Event page executed when a player object touches the current scene object</td></tr>
<tr><td>Parallel events</td><td>Event pages that keep executing in a loop</td></tr>
<tr><td>Appear events</td><td>Event pages that execute when a scene object appears</td></tr>
<tr><td>Leave touch</td><td>Event page executed when a player object leaves the current scene object after touching it</td></tr>
</table>

## Interface Generic Events

The trigger events that interface components have when executing events, have more conditions so the image cannot be cut in full.

![](img/Understand/005/003.png=600-)

<table>
<tr><td>Click</td><td>Event page executed when the mouse clicks on the current interface component</td></tr>
<tr><td>Mouseover</td><td>Event page executed when the mouse hovers over the current interface component</td></tr>
<tr><td>Mouseout</td><td>Event page executed when the mouse hovers over the current interface component</td></tr>
<tr><td>Display</td><td>Event page executed when the current interface component is displayed</td></tr>
<tr><td>WUnDisplay</td><td>Event page executed when the current interface component disappears</td></tr>
<tr><td>LeftDown</td><td>Event page executed when the left mouse button is pressed on the current interface component</td></tr>
<tr><td>LeftUp</td><td>Event page executed when the left mouse button ejects the current interface component</td></tr>
<tr><td>LeftDblclick</td><td>Event page executed when the left mouse button double-clicks on the current interface component</td></tr>
<tr><td>MouseMove</td><td>Event page executed when the mouse moves over the current interface component</td></tr>
<tr><td>RightDown</td><td>Event page executed when the mouse right click is pressed on the current interface component</td></tr>
<tr><td>RightUp</td><td>Event page executed when the right mouse button ejects the current interface component</td></tr>
<tr><td>RightClick</td><td>Event page executed when the right mouse button clicks on the current interface component</td></tr>
</table>
