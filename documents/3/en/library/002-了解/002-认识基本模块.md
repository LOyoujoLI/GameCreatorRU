## Preface

When we open the **database** the pop-up screen is related to the customization module, which covers most of the game's settings.

## World Settings

### Interface display

![](img/Understand/002/001.png=600-)

### Description

World settings can set the general settings of the game

For example: set the scene you enter after starting the game (the initial scene of the game), the total number of archives, the game's general sound settings, etc.

## Player settings

### Interface display

![](img/Understand/002/002.png=600-)

### Description

The player's own settings, such as the props and gold he owns.

### Avatar

### Interface display

![](img/Understand/002/003.png=600-)

### Description

The avatar (sprite) style of the scene object, walking, running and other action styles are also set by this.

## Standing avatar

### Interface display

![](img/Understand/002/004.png=600-)

### Description

You can put all the character's standing avatar here, and the standing avatar module also allows you to change the expressions, attributes and other settings of the current standing avatar.

## Animation

### Interface display

![](img/Understand/002/005.png=600-)

### Description

Some of the game's animation effects we can set through the animation module, through k-frames we can quickly create animations

## Dialog box style

### Interface display

![](img/Understand/002/006.png=600-)

### Description

In the game, the look of our dialog box is based on the dialog box style, if you want to change the look of the dialog box you can modify it directly here

## Interface

### Interface display

![](img/Understand/002/007.png=600-)

### Description

All the interfaces in the game come through the interface module, if you want to add a certain game interface, you can set it in the interface module.

Templates are generally preset with some common interfaces, you can directly modify the material of the preset interface to make your game exclusive interface.

**Note: Components and interfaces with security locks cannot be deleted, and components can be hidden if you do not want them to be displayed** ![](img/Understand/002/009.png)

## Props

### Interface display

![](img/Understand/002/008.png=600-)

### Description

A common prop setting in RPG, you can set whether the consumables, prices, etc., and through the command you can operate the props on the player to increase and decrease.
