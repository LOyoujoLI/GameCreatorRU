## Start

Commands are very common functions, and many logical relationships can be made through commands.

For example, execute a dialogue, determine whether the player has a certain prop or not, and whether the player has passed the level once.

## Use

Here is an example of displaying a piece of dialogue.

You can open the command page by right-clicking [event page](?file=002-了解/004-认识事件页), and then select the **Show Dialogue** command within the command page.
