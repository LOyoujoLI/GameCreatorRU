## Start

Scene objects can be created with the **Clone Object** command.

![](img/In-depth/018/001.png)

## Use the

### Specify the scene map and scene objects

Since the scene object is created by cloning an existing object, you need to specify the scene object as well as the scene object number.

The recommended way is to create a scene map specifically to hold the scene objects you want to clone.

## Destroying cloned objects

If you wish to delete the cloned object, you can use the **Destroy Cloned Object** command.

![](img/In-depth/018/007.png)

### Specify the object to destroy

You need to specify the cloned object to use this command properly, and the default provides the executor, trigger, and specified numbering method.

![](img/In-depth/018/008.png=600-)

#### Bind the clone object number

If you are not sure what the number of the clone object is, you can bind the clone object number when cloning the object.
