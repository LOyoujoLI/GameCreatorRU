## Start

The process of making games inevitably requires testing games, and as a game developer it is even more important to know how to use auxiliary functions to improve your testing and development efficiency.

## Test Mode

Find **Database -> World Settings ->Debug** and you can do different things in the game according to different options.

![](img/In-depth/012/001.png=600-)

### Show map grid obstacles

When checked, the obstacle areas in the map will be displayed in-game.

Red is for scene obstacles, orange is for object obstacles.

![](img/In-depth/012/002.png)

### Show object collision box

When checked the scene object shows the red rectangle outer box, when collision to the red rectangle outer box triggers the collision event page.

![](img/In-depth/012/003.png)


### Left mouse button to drag and drop objects

Check this box to drag and drop the object position with the left mouse button in the game.

![](img/In-depth/012/004.gif)

### Scroll wheel zoom lens

Check this box to zoom the scene lens with the mouse wheel in the game.

![](img/In-depth/012/005.gif)

## Disable, terminate event handling

### Disable

You can disable commands you don't want to execute by selecting the command and right mouse click.

![](img/In-depth/012/006.png)

![](img/In-depth/012/007.png)

### Terminate event processing

After executing the terminate event handling command, subsequent commands on the event page will not be executed.

![](img/In-depth/012/008.png)
