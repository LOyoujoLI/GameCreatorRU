## Start

Setting keys in GC is very simple, we can bind them with the **Keyboard Event** command.

![](img/In-depth/003/001.png)

## Binding keystrokes

Mouse click on the button, press any key on the keyboard and select the dropdown box to determine the trigger condition to bind successfully.

![](img/In-depth/003/002.png)

![](img/In-depth/003/003.png)

The life cycle determines how long the current command will be in effect.

![](img/In-depth/003/004.png)

## Trigger event page

The event page that is executed when a key is pressed.

![](img/In-depth/003/005.png=600-)

## Cancel key command

When we don't want this command to take effect, we can cancel it by checking **Record Listener** and then bind the variable, and then cancel it with the **Cancel Keystroke Event** command.
