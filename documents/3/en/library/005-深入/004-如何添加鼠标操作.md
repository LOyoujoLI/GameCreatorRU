## Start

Setting up a mouse action in GC is very simple, we can bind it with the **mouse event** command.

![](img/In-depth/004/001.png)

## Bind button

By selecting the dropdown box to determine the trigger condition, the life cycle can determine when the current command will take effect.

![](img/In-depth/004/002.png)

## Trigger event page

The event page that is executed when a key is pressed.

![](img/In-depth/004/003.png=600-)

## Cancel keystroke command

When we don't want this command to take effect, we can cancel it by checking **Record Listener** and then bind the variable, and then cancel it with the **Cancel Keystroke Event** command.

![](img/In-depth/004/004.png)

## Restricted to use on scenes only

When **Use on Scene Only** is checked, the mouse events of this command will only work on the scene map.
