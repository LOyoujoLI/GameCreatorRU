## Start

In GC, although we can make animation effects quickly with commands, sometimes we still want to be able to preset animation effects out directly.

This is when we can find the **Database -> Animation** module.

![](img/In-depth/010/001.png=600-)


## Operation

The animation is created in two main ways.

1、Import the material

2, K-frame (that is, select the track at the bottom of the animation module, the number at the top is the different number of frames, you just need to do different operations on the resources at different frames to generate the animation effect.)

In the gif, I have rotated the imported material.

![](img/In-depth/010/002.gif=600-)

## Image source

The image source is similarly numbered, and the number displayed in the upper left corner of the gallery is the ID of the image source.

![](img/In-depth/010/003.png)

During the animation process we can switch the source to be displayed by switching the source.

![](img/In-depth/010/004.gif=600-)

## Layer hierarchy

There is a layer hierarchy in the animation module, following the principle that children are controlled by their parent and the parent drives the children.

### Modifying axis points with layer hierarchy

One trick is to modify the axis of a layer by using the hierarchy.

The parent acts as the pivot point for the children

![](img/In-depth/010/005.gif=600-)

![](img/In-depth/010/006.png)

## Add a transition method to the animation

If you don't want the transition method of the animation to be too linear, you can modify the current transition method by right-clicking the shutdown button.

Please choose the specific transition method to use according to the actual situation.

![](img/In-depth/010/007.gif=600-)

## Nested animations

Different animations are nestable with each other.

The length of the embedded animation depends on the original length of the embedded animation and the total number of frames in effect for the current animation.

![](img/In-depth/010/008.gif=600-)

## Nested sound effects

Animations can also be embedded with sound effects, the duration of which depends on the original duration of the sound effect and the total number of frames in effect for the current animation.

![](img/In-depth/010/009.gif=600-)
