## Start

Use **event pages** to manage all **closures** of **game interface**.

The event pages will be familiar to you, you can just write them on the **Common Event** and call them afterwards.

The game interface is definitely closed using the **Close Interface** command.

So how to judge the interface? Once it comes to judging we must react to the fact that we can use **conditional divergence**.

The overall logic is sorted out: determine whether the specified interface is displayed by conditional divergence, and if so, close the interface.

## Determine the specified interface

Find the interface options through the conditional divergence, select the interface you want to judge, and then the drop-down box we choose **Front of the line**.

Why choose the front-most display? Because in the game there are multiple game interface at the same time, here I want to close the interface is the most front display, of course, the specific choice or you need to judge according to the actual situation.

## Close command

Next, just write close the specified interface within the conditional divergence.

In addition to the close interface Command I also wrote the termination event handling, why so? Because this event page is expected to manage the closing of all interfaces, and the termination event handling is added here to stop the execution of subsequent Command once the condition is met, reducing unnecessary calculations.
