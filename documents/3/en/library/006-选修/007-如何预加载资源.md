## Start

When playing games online, one important point is resource loading, for example, if the game starts before the image is successfully loaded, it will cause problems such as the image not being displayed during the player's play.

We can use the **preload resource** command to load the resource first, so that it will not cause embarrassing problems such as not displaying.

![](img/Optional/007/001.png)

## Use

Just directly specify the resource you want to preload.

![](img/Optional/007/002.gif=600-)

## Occupy process

Preload Command occupy the process execution until the resource is loaded before continuing with subsequent Command.

You can use **independent events** if you don't want the process to be occupied.

## Preload progress bar

The preload resource Command can be used with a slider component to create a progress bar effect.

Just select the specified component, followed by a drop-down box to select the property you wish to change.
