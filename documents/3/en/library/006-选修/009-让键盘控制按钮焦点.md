## Start

The **Set Button Focus** allows the keyboard to select buttons nicely, eliminating the need for the mouse to control them.

![](img/Optional/009/001.png)

The button focus has more parameters, which will be explained one by one.

<table>
<tr><td>Interface</td><td>Specify the interface where the component is located</td></tr>
<tr><td>Autofocus mode</td><td>All button components at the level below the root container are set to focus</td></tr>
<tr><td>Append button</td><td>Specify button component to set focus</td></tr>
<tr><td>Exclude buttons</td><td>Specify button components without focus</td></tr>
<tr><td>Skin</td><td>Style when in focus</td></tr>
<tr><td>Setting the focus index</td><td>where the initial focus is</td></tr>
<tr><td>Allows shortcut keys to exit current focus</td><td>Use ESC to quickly exit focus mode</td></tr>
</table>
