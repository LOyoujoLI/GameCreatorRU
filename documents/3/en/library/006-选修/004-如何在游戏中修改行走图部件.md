## Start

The **Modifying Avatar Parts** command allows you to replace an existing part of the specified object with another avatar part.

![](img/Optional/004/001.png)

## Use

After specifying the object that triggers this command, the selected existing part will be replaced with another avatar part.

Note: The object must originally have the selected part.
