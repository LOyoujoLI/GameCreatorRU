## Option setting judgment condition

You can enable different judgment features by clicking on the option settings.

![](img/GettingStarted/006/006.png)

### Default options

The default position for the focus point to focus on when the option appears.

Note: The default option style is based on **Database -> Dialog Style -> Focus Image**.

![](img/GettingStarted/006/007.png)

![](img/GettingStarted/006/008.png)

### Cancellation Occasions

When a cancelled occasion is set, you can close the current option by pressing the keyboard shortcut ESC and trigger the command within the set cancelled occasion option.

![](img/GettingStarted/006/009.png)


### Occurrence conditions

Occurrence conditions can be used to determine whether the current option is shown or hidden by matching the variable judgment.

About variables and the judgment can be seen in **[Recognizing variables](?file=file=004-基础/002-认识变量)**, where the judgment is similar to **[Conditional divergence](?file=004-基础/004-认识条件分歧)**

![](img/GettingStarted/006/010.png)

![](img/GettingStarted/006/011.png)
