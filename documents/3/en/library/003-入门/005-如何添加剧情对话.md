## Start

1. Find the place where you want the event page to be triggered, here is a scene object as an example. Right mouse button against the scene object at the click event to insert the open command page.

![](img/GettingStarted/005/001.gif=600-)

2. Find **Show Dialog** in **News** (If you can't find it, you can find it by searching! [](img/template tutorial/AVG-Starry Pupil/Starting/002/Search.png))

![](img/GettingStarted/005/002.png=600-)

![](img/GettingStarted/005/003.png=600-)

3. On the left we can set the character name for this conversation, the character avatar

![](img/GettingStarted/005/004.png=600-)

4, in the content area we can directly enter the content of the dialogue

![](img/GettingStarted/005/005.png=600-)

5, the top of the content area can be adjusted for the dialogue style, parameters. (The main operation here is to modify the color)

![](img/GettingStarted/005/006.png=600-)

![](img/GettingStarted/005/007.png=600-)

6, we can preview the effect in the game by clicking **Preview**

![](img/GettingStarted/005/008.png=600-)

![](img/GettingStarted/005/009.png=600-)

## Test

We can test the game directly by clicking on the menu **Test Game (shortcut F9)**.
