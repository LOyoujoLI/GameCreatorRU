## Start

The whole tutorial we have been able to make a simple case, finally you will make the game content published!

Of course, please also pay attention to the following points before publishing.

## Make sure there is a player object present

We need to make sure that we have a birth point for the player and that it is the game scene you want to appear after starting the game.

![](img/GettingStarted/015/001.png=600-)

## Whether the player can start the game properly

Usually the player does not start the game after opening the game, but enters the title screen. Make sure that the title screen has a button to start the game and commands.Make sure the title screen has a start button and commands.

![](img/GettingStarted/015/002.png=600-)

### Command to start the game

Actually the **Start Game button** is just a decoration, the real execution is the **Start Game** command.

We can find the command in the command page **System -> Start Game**.

![](img/GettingStarted/015/003.png)

## Test the game

Be sure to test the game once before you pack it to make sure it's okay.

![](img/GettingStarted/015/004.png)

## Start packaging and publishing

We can quickly package and release our game by using **Publish The Game**.

![](img/GettingStarted/015/005.png)

### Local version

Currently, we support windos and web package release. The Web requires you to build your own server environment before you can play.

![](img/GettingStarted/015/006.png)
### GC game platform

Support to publish the game package to GameCreator website, so that you can play online through GameCreator's official website.

![](img/GettingStarted/015/007.png)
