## Start

Switching game scenes is very easy, only one command is needed.

1 By clicking + you can create a new scene map.

In the gif I also performed a modify map level operation, this function can be very convenient to organize the order.

![](img/GettingStarted/010/001.gif=600-)

2 Right-click to select can perform operations such as changing the map name.

![](img/GettingStarted/010/002.png)

![](img/GettingStarted/010/003.png)


3 Next, right click on the command page where you wish to execute the command to insert

Find **Scene** in **Transfer Player** 

Note 1: Left mouse click to select the location of the scene object to reach after switching maps.

Note 2: Press and hold space to drag to move the preview window.

![](img/GettingStarted/010/004.gif=600-)

![](img/GettingStarted/010/005.png)
