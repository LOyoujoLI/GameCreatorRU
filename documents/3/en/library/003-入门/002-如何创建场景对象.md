## Create scene objects with styles

Scene objects can be created quickly by dragging and dropping the Scene Object option directly on the left.

![](img/GettingStarted/002/001.gif=600-)

## Creating an empty scene object

An empty scene object, as the name implies, does not come with a style and will not be displayed in the game, but will actually still exist.

![](img/GettingStarted/002/002.gif=600-)
