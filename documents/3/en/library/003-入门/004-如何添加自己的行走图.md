## Start

You can add and modify your own avatar via **Database -> Avatar**.

In the demo, an omnidirectional avatar is used, but it is possible to add a single-directional avatar.

![](img/GettingStarted/004/001.gif=600-)

## more

> The following content is an explanation of the common functions of the avatar module, if you don't quite understand it's okay, you can come to it a little.

### Direction

The maximum can support eight directions of avatar use.

![](img/GettingStarted/004/002.png)

### Omni-directional avatar

Just follow the arrows on the left to place different directions of the avatar.

![](img/GettingStarted/004/003.png=600-)

### Batch operation

The right mouse button on the avatar frame allows for some batch operations

#### Alignment

You can align the current direction and all directions of the avatar with the position operation.

![](img/GettingStarted/004/004.gif=600-)

#### Displacement

You can displace the current direction, all direction avatar, and support displacement on the original (relative).

![](img/GettingStarted/004/005.gif=600-)

### Movement expressions

When you want to make different actions for the avatar, you can use the action expressions function, which supports custom creation and import.

![](img/GettingStarted/004/006.gif=600-)

### Part List
