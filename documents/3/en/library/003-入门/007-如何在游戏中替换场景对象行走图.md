## Start

1, through **object behavior** command can do a lot of operations on the scene object in the game, here to replace the avatar as an example.

2、Find **Set Object Behavior** in **Scene Object**

![](img/GettingStarted/007/001.png=600-)

3, through here you can choose the need to modify the scene object, here to scene object 1-NPC for example.

![](img/GettingStarted/007/002.png=600-)

4. Click **change avatar** to select the style of avatar you wish to change.

![](img/GettingStarted/007/003.png=600-)

![](img/GettingStarted/007/004.png=600-)

5, here I replaced the avatar with an archer.

![](img/GettingStarted/007/005.png=600-)

![](img/GettingStarted/007/006.png=600-)

6. Simply write an option command.

![](img/GettingStarted/007/007.png)
