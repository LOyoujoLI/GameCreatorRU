## Create a new layer

The specified layer can be created by right-clicking on the layer area.

![](img/GettingStarted/011/001.png)

## Using blocks to draw maps

> There are two types of blocks, normal blocks and automatic components. The difference is that automatic components can automatically replenish blocks with different angles, while normal blocks require you to replenish blocks with different angles manually.

### Basic operation

When the block layer is selected, the drawing tools area will appear and you can draw the scene map according to the different tools

![](img/GettingStarted/011/002.png=600-)


<table>
<tr>
<td>Brush</td> <td>Can draw blocks to the scene map</td></tr>

<td>Rectangle</td> <td>Selects an area on the scene map to fill with the block</td></tr>

<td>Fill</td> <td>Block fill all areas of the scene object</td></tr>

<td>Erase</td> <td>Clear the specified blocks</td></tr>

<td>Select</td> <td>Select an area and right mouse click to cut, copy, or delete</td></tr>

</table>

### Shortcuts

#### Right mouse button selection

The right mouse button can select the block at the current location on the current scene map, if you long press the right mouse button you can multi-select.

![](img/GettingStarted/011/005.gif=600-)

#### Spacebar drag and drop

Long press space to drag and drop to preview the map.

![](img/GettingStarted/011/006.gif=600-)

#### Adjusting more parameters

More useful parameters can be adjusted by turning off drawing mode.

Such as zoom, offset, tilt, overlay method, material.

![](img/GettingStarted/011/007.gif=600-)
