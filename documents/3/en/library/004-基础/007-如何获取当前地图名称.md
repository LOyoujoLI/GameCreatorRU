## Start

First let's analyze our requirements.

We want to get the **scene map name** within the game.

When we want to get the content of the game, we should think that we can use **game variables** to do so.

And to get the name of the map, then it should be **String**, because the map name is neither a number nor a switch.

## Get the map name

Let's just open the string variable directly and check the game string.

![](img/Basic/007/001.png)

We are looking to get the name of the map on **Scene**, so we find **Scene** and just select the name.

![](img/Basic/007/002.png)

Then the value of this string variable is the name of the current map.

![](img/Basic/007/003.png)
