## Logic finishing

### Where is the title screen executed to be displayed?

I want the buttons to have transition animations when the title screen is displayed, so first we need to find out where the title screen is executed to be displayed.

The **Common Event->System Events->Start the Game** title screen is displayed here with the **Show Interface** command.

![](img/Basic/008/002.png=600-)

### Where do I write the directive?

I want the animation effect I made to be executed every time the title screen is displayed, so we have to write the animation effect where the command will be triggered every time the title screen is displayed.

**Database->Interface->Title->Root Container->Display** Every time the interface is displayed the interface root container is bound to be displayed, so we can execute the animation effect directive in the display time trigger of the root container.

![](img/Basic/008/003.png=600-)

## Caution

Since this tutorial only has a transition effect, there are no other Command after the transition so there is no need to write **wait**.

Assuming that other Command need to be added after the transition of this event page, you need to add wait, otherwise the transition effect will start executing the subsequent Command before it is finished.
