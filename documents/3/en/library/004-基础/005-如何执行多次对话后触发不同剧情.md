## Start

We can count the current number of conversations by **numeric variables**, and then execute different episodes when the number of conversations reaches a certain number by **conditional divergence**.

## Counting the number of conversations

We just need to perform the mathematical addition of **numeric variables**.

Since we are counting the number of conversations, here we perform **+1** so that every time the button is clicked, then the variable will be added to the original by +1.
