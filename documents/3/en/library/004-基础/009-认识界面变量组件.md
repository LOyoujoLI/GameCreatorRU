## Start

When we need to display the variable resident in the interface, we can use the corresponding variable component to do so.

## Numeric

The display of numeric values we can achieve with **Numeric Variable Component**, **Game Numeric Component**.

![](img/Basic/009/001.png)

![](img/Basic/009/002.png)

### Use

Simply drag the component out and use it. With the properties on the right side you can change the style and the source to get the value.
