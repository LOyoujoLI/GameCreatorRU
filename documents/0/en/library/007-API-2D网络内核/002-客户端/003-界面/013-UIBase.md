## Details

### EVENT_COMPONENT_CONSTRUCTOR_INIT
###### EVENT_COMPONENT_CONSTRUCTOR_INIT : string;
[Static] Events: Events dispatched during initialization of component construction<br>
This event allows you to listen for all component initialization so that you can append logic<br>
>EventUtils.addEventListenerFunction(UIBase, UIBase.EVENT_COMPONENT_CONSTRUCTOR_INIT, (uiComp: UIBase)=>{<br>
>&nbsp;&nbsp;// to do<br>
>}, this);<br>
>


### ON_VISIBLE_CHANGE
###### ON_VISIBLE_CHANGE : string;
[Static] Event: Appearance or disappearance due to emergence conditions (VISIBLE CHANGE)<br>
>ui.on(UIBase.ON_VISIBLE_CHANGE,this,func);<br>
>


### commandInputMessage
###### commandInputMessage : any;
Submit player input information Data type:any[] | Callback<br>
Used to load submitted player input values to be received by the event page (waiting for player submissions), either as a fixed array of data or as an array of data to be returned in the callback function<br>
For example, if the control has a click event and the click is submitted with parameters, the input value will be received in the event page after submission
