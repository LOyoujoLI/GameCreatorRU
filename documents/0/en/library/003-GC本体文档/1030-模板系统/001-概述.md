# Template System Overview
A template is an entire game project packaged as a template to share or sell for others to use. <br> 
Templates can be made based on GC kernel for secondary development, or written from scratch using Empty project. <br> 
The template store is not yet open for uploading templates made by third parties, please be patient. <br> 
>The GC visual editor is written and made with Empty as GC2D kernel and GC2D network kernel<br> 
>And "ARPG - Book of black" is another ARPG single-player game template generated by secondary development based on GC2D kernel.

# About the mechanism of template production and purchase
If a template is based on another template, the author will be required to have purchased the other parent template when purchasing that template. <br> 
Also supports settlement of simultaneous purchases. <br> 
> For example, if author C uses "ARPG - Book of black" to develop a second time to add N additional module features and sell it again, the purchaser will require to have purchased "ARPG - Book of black". <br> 
> For example, if template B was developed using template A, and someone developed template C using template B, the purchase of template C will require the purchase of both B and A. (Up to the root)
