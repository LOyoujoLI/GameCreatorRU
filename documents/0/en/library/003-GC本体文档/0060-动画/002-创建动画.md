# Create animation


## Go to the animation editor
1-To open the database, click the database icon or shortcut key F10 to open the database<br>
<img src="img/new/3-2.png" /><br><br>
2-Switch to the animation editor<br>

## Data List
The animation data is managed uniformly using numbering.<br>
Divided into 16 categories, the maximum value can be changed in each category in order to adjust the range of available animations.<br>
<img src="img/2/动画/1-1.png" /><br>

## Select animation data
Left click on the specified animation to open the animation data for editing.
