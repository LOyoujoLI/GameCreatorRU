# Importing material

## Import My Resources
To import using your own image clips, please use [Import My Resources](?file=003-GC本体文档/0008-资源/002-导入资源) first., import the material to the project.

## Importing sequence frame animation material
Click to add new animation material.<br>
<img src="img/2/动画/1-3.png" /><br>
Select a picture or multiple picture clips and click OK to finish importing. When selecting one, you can choose to cut the picture and guide it into multiple cut pictures. <br>(The original image will not be cut, only the cut part will be shown when the game is running according to the setting)<br>
<img src="img/2/动画/1-4.png" /><br>
