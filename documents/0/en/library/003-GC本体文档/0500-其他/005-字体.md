# font
GameCreator allows multiple fonts to be used in the same game at the same time. <br> 
Allows font files to be imported into the game and published with the fonts when they are published. <br>

## Default Font
The game default font can be changed in [Settings]. <br> 
The setting will affect the default value for text without a font set and when a new text component is created. <br> 
It will not affect text for which the font has been set.

## Importing Fonts
After importing fonts you can choose whether to load them at engine initialization<br> 
or let the project layer decide when to load them itself, using <b>[AssetManager](?file=006-API-2D单机内核/002-客户端/012-AssetManager).preloadFonts</b>. <br> 
The imported fonts are stored in the asset/font folder,<br> 
and the fonts are used according to the imported file name, as in the case of the font below, the font is set to ggg. <br>
