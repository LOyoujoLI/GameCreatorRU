# Display Hierarchy
GameCreator provides a default display hierarchy arrangement. <br>
Advanced developers can adjust the display hierarchy by overriding the method. <br>
<img src="img/new/0.png" /><br>
## Show hierarchical list
<table>
 <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Stage</td> <td> 
      The ultimate root container for all display objects, with a "game layer" added inside. 
    </td>
 </tr>
 <tr>
    <td>GameLayer</td>  <td>
      Inside the game layer, "scene layer", "image layer" and "interface layer" are added in order.
      <br>
    </td>
 </tr>
 <tr>
    <td>SceneLayer</td>  <td>
      The scene layer is used to place scenes (display objects), usually there is only one scene.
    </td>
 </tr>
 <tr>
    <td>ImageLayer</td>  <td>
      The image layer is used to display pictures, animations, videos, etc. relative to the game window screen coordinate system.
    </td>
 </tr>
 <tr>
    <td>UILayer</td>  <td>
      The interface layer is used to display the interface.
    </td>
 </tr>

 </table>
