# Resource Overview
Various performance effects within the game require calling different game resources in order to use them.<br>
After preloading the resource, the resource can be used to immediately act and render, otherwise it will be dynamically loaded and the correct effect will not appear until it is loaded (PC games have a faster loading time because the resource is local).<br>
==(If used on mobile, the image size should not exceed 4096x4096)==<br>

## Resource categories and supported formats
==[Basic Resources]==<br>
Images: png, jpg, jpeg <br>
Text book: txt, json<br>
Audio: mp3, ogg<br>
Video: mp4, webm<br>

==[Advanced Resources]==<br>
Advanced resources consist of a combination of corresponding production data and a variety of basic resources<br>
「Scenes:Scene」 Image resources that may contain several scene objects, layer references<br>
「Scene object:SceneObject」  May contain a avatar, several animations of the mount and the interface of the mount<br>
「Avatar:Avatar」  May contain pictures, quoted parts (avatar)<br>
「standing avatar:StandAvatar」 May contain pictures, quoted parts (standing avatar)<br>
「Animation:Animation」 May contain pictures, audio, quoted animation (animation)<br>
「Interface:GUI」 May contain pictures, avatar, standing avatar, animations<br>
「Dialog box style:Dialog」 May contain pictures, animations (SKIP logo), voice audio<br>
「Event Page:CommandPage」 May include pictures, animations, avatar, standing avatar, interfaces, dialog box styles, dialogue avatars, audio, etc.<br>

For example, if you modify an image in PhotoShop, which is already in use within GC, the modification will be saved and displayed in GC with real-time updates without re-importing.<br>
