### Show picture
Just quote the address directly, it will be preloaded internally.
>var bmp = new UIBitmap();<br>
bmp.image = "asset/image/abc.png";<br>
bmp.x = 300;<br>
bmp.y = 200;<br>
stage.addChild(bmp);<br>

Prioritize preloading and display when preloading is complete.
>AssetManager.loadImage("asset/image/abc.png", Callback.New((tex: Texture) => {<br>
&nbsp;&nbsp;var bmp = new UIBitmap;<br>
&nbsp;&nbsp;bmp.texture = tex;<br>
&nbsp;&nbsp;bmp.x = 300;<br>
&nbsp;&nbsp;bmp.y = 200;<br>
&nbsp;&nbsp;stage.addChild(bmp);<br>
}, this));<br>
