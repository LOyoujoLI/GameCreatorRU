# Importing Resources
There are currently three ways to import resources into the current game project:<br>
<table>
 <tr>
    <td><b>Operation</b></td>  <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Drag and drop files into the workspace</td>  <td>Importing files into the current directory</td>
 </tr>
 <tr>
    <td>Right mouse click on the blank space and select <u>Importing new resources</u></td>  <td>Importing files into the current directory</td>
 </tr>
 <tr>
    <td>Manually copy the resources to the asset folder in the project directory</td> <td>GC Explorer is automatically refreshed in</td>
 </tr>
 </table>

### Way 1: Open the appropriate explorer and drag and drop files into the work area<br>
 Open F5-Material Library (for audio resources, open F6-Media Library)<br>
 <img src="img/new/1_10.png" /><br>
 Drag and drop files into the workspace<br>
 <img src="img/new/1_11.png" /><br>
 Import completed<br>

### Way 2: Right mouse click on the blank space and select <u>Importing new resources</u><br>
 <img src="img/new/1_12.png" /><br><br>

### Way 3: Manually copy the resources to the asset folder under the project directory<br>
 Open the directory where the current project is located:<br>
 1.Menu->Open Project Location<br>
 2.Copy or move resources to the asset folder<br>
 > GC Explorer will refresh automatically
