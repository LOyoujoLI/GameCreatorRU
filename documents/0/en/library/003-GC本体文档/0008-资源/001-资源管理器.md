## Search for resources
Search for the keyword of the file name to list all files containing the keyword, support fuzzy search (symbol *)
>For example, search for a*c can search for abc or a123c and so on.

## File Relative Path
Open the menu with the right mouse button to see「File Relative Path」。<br>
OK will be automatically copied to the clipboard, all the resources used in the game are basically using relative paths,==<b>Use slash for paths'/' Instead of backslash '\\'</b>==。<br>

## Rename files/folders
 Select the file or folder and use the right mouse button to open the menu to see「Rename」。<br>
 ==Rename can modify the file format, if the file format is changed, it will be modified, if the format is not entered, the original format will be retained.==

## Delete files/folders
 Select the file or folder and use the right mouse button to open the menu to see「Delete」。<br>
 ==Currently deleting files or folders does not go to the Recycle Bin, so delete with caution.==

## File/folder properties
 Select the file or folder and use the right mouse button to open the menu to see「Properties」。<br>
 It is easy to see the basic information of the files as well as the number of subfiles and the total occupied size under the folder.<br>

## Create a new folder
 When the mouse clicks on a blank area, use the right button to open the menu to see「Create a new folder」。<br>
 After creating a new folder, a selection box for naming will pop up, and you can see the new folder after naming it.

## Refresh
When the mouse clicks on a blank area, use the right button to open the menu to see「Refresh」。<br>
 Resource management uses a cached file design（asset/assetList.json），When changing resources through a non-GC internal resource manager,<br>
 This feature is usually not necessary because the system will refresh automatically.

## Copy the current full file path
 When the mouse clicks on a blank area, use the right button to open the menu to see「Copy the current full file path」。<br>
 Store the copy result in the clipboard, which may be used usually when writing scripts, to facilitate listing all files in the specified directory.<br>
 For example, there is no API on the WEB side to get all the files under the specified folder, which can be written directly to the code by this method.
