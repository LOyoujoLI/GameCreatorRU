# Creating a avatar


## Go to the avatar editor
1-Open the database, click the database icon or shortcut key F10 to open the database<br>
<img src="img/new/3-2.png" /><br>
2-Switch to the avatar editor<br>

## Data List
The avatar data is managed uniformly using numbers.<br>
Divided into 16 categories, the maximum value can be changed in each category in order to adjust the range of available avatars.<br>
<img src="img/2/行走图/1-3.png" /><br>
<img src="img/2/行走图/1-4.png" /><br>
<img src="img/2/行走图/1-5.png" /><br>


## Select avatar data
Left mouse click on the specified avatar to open the avatar data for editing.

    Mouse operation: right mouse button to select <u>Clear</u>
    </td>
 </tr>
