# Importing material

## Import My Resources
To import using your own image material, please use [Import My Resources](?file=003-GC本体文档/0008-资源/002-导入资源) first to import the material under the project.

## Importing images as avatar
Click to add a new avatar.<br>
<img src="img/2/行走图/1-7.png" /><br>
Select one or more avatar and click OK. When selecting one, it can be a single map or an omnidirectional avatar.<br>
<img src="img/2/行走图/1-9.png" /><br>
Once the import is complete, there are corresponding frame maps for each of the four directions, all with 4 frames each.<br>
<img src="img/2/行走图/1-17.png" height=200 />
<img src="img/2/行走图/1-18.png" height=200 />
<img src="img/2/行走图/1-19.png" height=200 />
<img src="img/2/行走图/1-20.png" height=200 /><br>
The unchecked all-direction avatar means that only the currently selected direction is imported, and there is only one frame map, so the horizontal and vertical cut is 1.<br>
<img src="img/2/行走图/1-16.png" /><br>
