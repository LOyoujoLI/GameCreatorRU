# Scene object prototype
A scene object prototype is a predefined class of objects consisting of == public modules== and == pre-installed scene object modules==. <br> 
Each scene object has its prototype, and the prototype has the characteristics that the object has. (For example, Xiaoming's prototype is a human, while Wangcai's prototype is a dog)<br>

## Open the scene object prototype editor
Select the scene object prototype at the bottom left of the editor, right mouse button to bring up the menu - select <u>Edit</u>, or add a new preset.



## Public Modules
A public module is a common feature shared by all classes of prototypes that share common characteristics, consisting of: ==properties==, ==display objects==, ==bound scripts==, ==editor previews==<br>

#### Client binding class
 Objects created in the game will be created based on the actual class. <br> 
 Client-side scene objects must inherit from SceneObjectCommon<br> 
 so that all objects in the client that are instances of that prototype are instances of that class.

#### Server binding class
 ==Only the web version has this property==<br> 
 Objects created in the game are created based on the actual class. <br> 
 Server scene objects must inherit from SceneObjectCommon<br> 
 so that all objects on the server side that are instances of that prototype are instances of that class.

## Prototyping

#### Configuration items
Classes that can be used to turn on or off emergent conditions, status pages, and trigger events so that different prototypes have different configurations of scene objects.

#### Settings Preview
You need to set the display objects in the set preview list to preview (right mouse button menu to set). <br> 
The list will automatically list the display objects in the "Public Module" or "Pre-installed Module". <br> 
Once set, the instances of this class prototype will be listed in the bottom left of the editor, making it easy to drag and drop to the map to generate scene objects directly. <br>

#### Pre-installed modules
Set the pre-installed modules for the prototype, which will be automatically installed when the prototype is dragged onto the scene.<br>
