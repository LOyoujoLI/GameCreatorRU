# Customized player settings
Players are in-game players, each of which corresponds to a player identity (there is only one player in the standalone kernel). <br> 
The player attributes are edited by the visual editor and implemented via scripting,<br> 
so that the game template can visually adjust the relevant player parameters, and each player has these attributes once set. <br>
>For example, the "Feather of cyan" template has preset gold and backpack attributes, and the player has gold and backpack attributes.<br> 
These attributes are custom attributes instead of system attributes because not every game has gold and backpack settings, and different games have different settings.

## Open the Player Properties Editor
Menu -> Custom Editor -> Player Attribute Editor (ALT+2)<br>

## Edit Player Attribute Panel
The methods for making the editor interface visually in the custom editor are the same, refer to [[Custom Editor Panel](?file=003-GC本体文档/1010-自定义编辑器/003-自定义编辑面板)]。

## Archive auto-association
Player attributes are stored at the time of archiving the current player attributes (except for configuration-only variables), and the producer does not need to write additional logic for archiving this data.
