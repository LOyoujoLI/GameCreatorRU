# Custom Data Structures
Custom data structures are generally used in conjunction with custom attributes. Some attributes are not of a single type, such as a player's backpack, <br> 
which records information about props, numbers, and equipment that the player has, rather than referring to a single value, and such compound cases can be made using custom data structures. 
> Similar to attribute types: Object instead of primitive number, string, boolean types

## Open the custom data structure editor
Menu -> Custom Editor -> General -> Custom Editor -> Data Structure Edit (ALT+0)<br> 
or can be edited when custom data structure is selected.

## Creating custom data structures
Define data structures to be managed uniformly using numbering. <BR> 
Adjust the range of available data structures by changing the maximum value. <BR>

## Interface for editing data structures
The methods for making the editor interface visually in the custom editor are the same, refer to [[Custom Editor Panel](?file=003-GC本体文档/1010-自定义编辑器/003-自定义编辑面板)]。

## Visualization of populated data
In the editor call to the component of this data structure, click on it and the corresponding panel will pop up, fill in the data.

## Proprietary variable properties: quick actions
Display and manipulate directly from outside, without manipulating inside the pop-up data structure window, and decide the arrangement according to the variable name order.

## Automatic system generation of API
The system automatically generates a DataStructureRuntime.api file based on all the current custom modules,<br> 
which is only used in the current memory and not the actual hard disk file, and the API is not available if you use a third-party IDE. <br> 
The corresponding data structure class is accessed by using ==DataStructure\_DataStructureName==, and the class definition is automatically generated based on pre-defined variables and their types, default values, etc. <br>
<img src="img/2/自定义/13.png" />

## Create a data structure data
A pre-defined data structure can be treated as a normal Object type and supports a visual panel to edit it. <br> 
Creation can be done directly using the new method: ==var a = new DataStructure\_DataStructureName==.
