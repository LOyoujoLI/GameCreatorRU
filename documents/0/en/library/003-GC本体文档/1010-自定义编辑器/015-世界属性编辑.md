#  World Properties Editor
The world is the only world in the game, and the world is the world common to all players (there is only one player in the single player kernel). <br> 
The world properties are edited through the visual editor and implemented through scripts or events so that the game template can visually adjust the relevant world parameters. <br> 
>For example, the "Feather of cyan" template has preset parameters such as sound effects, game start events, etc. so that the author can visually adjust these parameters

<img src="img/t10.png" /><br>

## Open the World Properties Editor
Menu -> Custom Editor -> World Property Edit (ALT+1)<br>

## Edit World Properties Panel
The methods for making the editor interface visually in the custom editor are the same, refer to [[Custom Editor Panel](?file=003-GC本体文档/1010-自定义编辑器/003-自定义编辑面板)]。

## Archive auto-association
World properties will store the current world properties at the time of archiving (except for configuration-only variables), and the producer does not need to write additional logic for archiving this data.
