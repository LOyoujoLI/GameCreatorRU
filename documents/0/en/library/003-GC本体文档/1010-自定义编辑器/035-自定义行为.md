# Customized Behavior
Support making custom [Scene Object Behavior](?file=003-GC本体文档/0020-场景对象/006-场景对象行为编辑) and preview the playback effect in real time. <br>
The following are custom object behavior commands, except for the change avatar.<br>
<img src="img/t26.png" />

## Open the custom behavior editor
Menu -> Custom Editor -> Behavior Settings Editor<br>

## Interface for editing behavior
The methods for making the editor interface visually in the custom editor are the same, refer to [[Custom Editor Panel](?file=003-GC本体文档/1010-自定义编辑器/003-自定义编辑面板)]。

## Realization behavior
Way 1: Create class to inherit from [SceneObjectBehaviors](?file=006-API-2D单机内核/001-通用/024-SceneObjectBehaviors) in order to implement the logic<br> You can refer to "RPG - Feather of cyan" template inside the ProjectSceneObjectBehaviors.ts<br><br>

Way 2: The behavior can be written inside the CustomSceneObjectBehavior module. <br>
>module CustomSceneObjectBehavior {<br>
&nbsp;&nbsp;/\*\*<br>
&nbsp;&nbsp;&nbsp;\* Custom Behavior-52 52 means the custom behavior corresponding to the number 52<br>
&nbsp;&nbsp;&nbsp;\* (p1, p2, p3 are parameters that need to be modified to the actual custom behavior corresponding to the parameters and types, in the order preset in the behavior editor)<br>
&nbsp;&nbsp;&nbsp;\* @param soBehavior Behavior Actuator<br>
&nbsp;&nbsp;&nbsp;\*/<br>
&nbsp;&nbsp;export function f52(soBehavior: SceneObjectBehaviors, p1: number, p2: number, p3: string): void {<br>
&nbsp;&nbsp;&nbsp;&nbsp;// to do<br>
&nbsp;&nbsp;&nbsp;&nbsp;<br>
&nbsp;&nbsp;}<br>
}<br>
