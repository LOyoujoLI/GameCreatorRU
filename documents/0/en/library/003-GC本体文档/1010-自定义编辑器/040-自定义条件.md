# Customized Conditional Divergence
In addition to the several condition divergence categories given by the system, custom condition divergence is also supported. <br> 
Different items (templates) may have different custom condition divergences. <br> 
Custom conditions will also appear in =="Game Switch"==. <br>
> For example, "RPG - Feather of cyan" provides several custom conditional divergences to make more determinations related to this template

<img src="img/2/自定义/2.png" /><br>

## Open the Custom Conditional Divergence Editor
Menu -> Custom Editor -> Conditional Divergence Settings<br>

## Interface for editing custom conditional differences
The methods for making the editor interface visually in the custom editor are the same, refer to [[Custom Editor Panel](?file=003-GC本体文档/1010-自定义编辑器/003-自定义编辑面板)]。
