# Custom Trigger Event Categories
Custom trigger event categories are used to define what trigger events are available for producers to edit events under these event pages. <br> 
Trigger events can have categories on scenes, scene objects, and interfaces. <br> F
or example, an event that is triggered when the mouse clicks on a scene object:<br> 
And where the event is executed or what conditions are met is something that needs to be designed and scripted. <br> 
Usually the various game templates use this feature to pre-set some of the categories that trigger events so that the person using the template can use them directly when they open the template. <br>

 <tr>

 </tr>
 <tr>
    <td>Scene object events</td> <td>Binding trigger events associated with scene objects, such as click events and touch events of scene objects</td>
 </tr>
 <tr>

## Implementing triggered scene events
Event page storage: Scene::customCommandPages, recording the relevant event pages by index. <br> 
Standalone related code example 1: 
>// Get the first event page of the scene: 1-enter event, indexed by ID-1, so 0 is used here<br>
var sceneCmdTypeIndex = 0;<br>
// Get Event Page<br>
var commandPageInScene = Game.currentScene.customCommandPages[sceneCmdTypeIndex];<br>
// Get the number of commands in the event page<br>
var inSceneCmdLength = commandPageInScene.commands.length;<br>
// If the number of commands is greater than 0, execute<br>
if (inSceneCmdLength > 0) {<br>
&nbsp;&nbsp;// Get event triggers: triggered by the player, the executor is also the player<br>
&nbsp;&nbsp;var commandTriggerInScene = Game.player.sceneObject.getCommandTrigger(CommandTrigger.COMMAND_MAIN_TYPE_SCENE, sceneCmdTypeIndex, Game.currentScene, Game.player.sceneObject);<br>
&nbsp;&nbsp;// Listening for one event execution, the<br>
&nbsp;&nbsp;EventUtils.addEventListener(commandTriggerInScene, CommandTrigger.EVENT_OVER, Callback.New(() => {<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;// Processing when event execution is complete<br>
&nbsp;&nbsp;}, this, []), true);<br>
&nbsp;&nbsp;// Start executing events<br>
&nbsp;&nbsp;commandPageInScene.startTriggerEvent(commandTriggerInScene);<br>
}
