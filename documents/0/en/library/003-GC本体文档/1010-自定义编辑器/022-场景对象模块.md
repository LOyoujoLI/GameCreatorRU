# Scene Object Module
A scene object module is a component that can be attached to a scene object. <br> 
Different modules have different characteristics, and when attached to the module it will have the characteristics of that module. <br> 
>For example, a shadow module has a shadow when attached.

## Open the Player Properties Editor
Menu -> Custom Editor -> Scene Object Module (ALT+2)<br>

## Creating custom object modules
Define modules to be managed uniformly using numbering. <BR> 
Adjust the range of available modules by changing the maximum value. <BR>
