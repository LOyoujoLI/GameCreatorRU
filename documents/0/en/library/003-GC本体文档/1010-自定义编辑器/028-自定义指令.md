# Customized commands (orders)
Users can customize appending event commands: visualize the editor for making event commands, and write the associated logic using scripts. <br> 
>For example, the "ARPG - Book of black" template appends some custom commands:<br>
<img src="img/t59.png" /><br>

## Open the custom command editor
Menu -> Custom Editor -> Command Settings Editor<br>

## Interface for editing commands
The methods for making the editor interface visually in the custom editor are the same, refer to [[Custom Editor Panel](?file=003-GC本体文档/1010-自定义编辑器/003-自定义编辑面板)]。

## Event Page Preview Mode
Allows real-time preview effects within the editor via scripts. <br> 
>For example, the real-time preview effects of commands such as Show Image, Show Video, etc. in the "Eyes of star" template are implemented in "Editor Preview". <br> 
>==This means that commands made by advanced developers are also available for real-time preview effects==.


##### Interface layer and image layer
 >Interface layer is used to display and manipulate the interface, and the Image layer is used to display and manipulate images/animations/standing avatar/videos, etc. <br>
You need to toggle the corresponding layer selected to allow operation.

##### Bifurcation Preview
>Real-time preview of the different conditions (routes) in the command without having to start the game to test the effect.

##### Preset Categories
>Resources from the preset categories can be dragged into the main stage to generate the corresponding commands.
