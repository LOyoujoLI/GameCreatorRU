## Introduction
In GameCreator, you can make different types of games according to the template of the corresponding type. The template provides the functions of the corresponding game type, and the author just needs to master the usage of that template function.

## Select a template

We need to log in and click **New Project** project to select the project template.

![](img/005.png=600-)

Select the template you want in the **Template Store**.

![](img/006.png=600-)

Finally, just create the project at **My Template**.

![](img/008.png=600-)

### Create project based on Development Core

### Post-creation projects
The finished project is stored in the project path you chose before creation, and you can also open the project directory in the editor by [Menu] - [Open Project Location].<br>
