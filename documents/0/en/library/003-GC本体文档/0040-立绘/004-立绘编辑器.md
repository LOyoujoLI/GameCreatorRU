## Importing material
 Click to add new standing avatar ([How to import my resources](?file=003-GC本体文档/0008-资源/002-导入资源)).<br>
 <img src="img/2/立绘/2-2.png" /><br>
 Select one or more standing avatar pictures and click OK to finish importing. When selecting one, you can choose to cut the picture and guide it into multiple cut pictures. <br>(The original picture will not be cut, only the cut part will be shown when the game is running according to the setting)<br>
 <img src="img/2/立绘/2-4.png" /><br>

### Object manipulation tools
<table>
 <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Select the object</td> <td>Left mouse click on the corresponding image, possibly a frame map/component</td>
 </tr>
 <tr>
    <td>Move selected object</td>
    <td>
        Keyboard control: Arrow keys to move 1 pixel or SHIFT+direction keys to move 10 pixels<br>
        Mouse operation: Drag and drop the object<br> 
        Property panel operation: Move the object by setting the x and y in the property panel
    </td>
 </tr>
 <tr>
    <td>Rotation</td>
    <td>
    Mouse operation: Drag and drop the black circle in the lower right corner of the object selection box<br> 
    Property panel operation: Set the rotation degree in the property panel by
    </td>
 </tr>
 <tr>
    <td>Scaling</td>
    <td>
    Mouse operation: Drag and drop the rectangular block of the object selection box, where the rectangular block at the bottom right will be scaled isometrically when SHIFT is held down<br> 
    Property panel operation: By setting the width/height/width zoom ratio/height zoom ratio in the property panel
    </td>
 </tr>
 </table>

## Frame Console

### Sequence frame operation
<table>
 <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Select the frame map</td> <td>Left mouse click on the frame to select it, and the workspace will switch to the current frame display<br>If there are parts,== their parts will also display the pattern for that frame corresponding to that action and that direction== (if there are not enough frames, the pattern will be taken)</td>
 </tr>
 <tr>
    <td>Insert frame</td> <td>Right click on frame image to bring up menu: insert frame image before current position, same as [Import material](#Import material)</td>
 </tr>
 <tr>
    <td>Move frames</td> <td>Right click on the frame map to bring up the menu: choose <u>Move left</u> or <u>Move right</u> to move the frame map to a different position, or you can ==change it by shortcutting ALT+left and right arrow keys==</td>
 </tr>
 <tr>
    <td>Copy Frame</td> <td>Right click on the frame map to bring up the menu: select <u>Copy</u> ==Copy via shortcut CTRL+C==</td>
 </tr>
 <tr>
    <td>Paste frame</td> <td>Right click on the frame map to bring up the menu: select <u>Paste</u> ==Paste via shortcut CTRL+V==<br>Paste will paste the previously copied frame map behind the currently selected frame</td>
 </tr>
 <tr>
    <td>Delete</td> <td>Right-click on the frame map to bring up the menu: select <u>Delete</u> ==delete via shortcut CTRL+D==</td>
 </tr>
 <tr>
    <td>Align the current frame coordinates in the current direction</td> <td>Set the other frame maps in the current direction to the coordinate position of the current frame uniformly</td>
 </tr>
 <tr>
    <td>Align all directions to the current frame coordinates</td> <td>Set the frame map in all directions to the current frame coordinates position uniformly</td>
 </tr>
 <tr>
    <td>Batch shift</td> <td>All sequence frames in the current direction (or all directions) can be set to a specified coordinate point or relative offset by a number of pixels<br>Relative: when checked, all sequence frames are shifted relative to their current coordinate point<br>All directions: when checked, this will work on all directions instead of all frames in the current direction</td>
 </tr>
 <tr>
    <td>Properties</td> <td>Open the Edit Frame Properties panel for editing, refer to the table below for details</td>
 </tr>
 </table>

 For imported images, select Properties in the right-click menu of the selected frame to bring up the edit window, where you can adjust the position and size of the crop. <br>

## Expression Management
For managing the expressions of this stand
<table>
  <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
  </tr>
   <tr>
   <td>Select the expression</td> <td>Left mouse click to select the specified expression, go to edit and view the frame map of that expression (including the frame map under all directions)</td>
   </tr>

   <tr><td>Add an emote</td><td>Select an added emote from the emote list<br>The emote list is managed by number-name unification, and the standing avatar may have one or more emotes<br>The meaning of the emote within the game depends on the specific item</td></tr>

   <tr><td>Copy Emoji</td><td>Copy the currently selected Emoji sequence frame set to the clipboard|==CTRL+C==</td></tr>

   <tr><td>Paste emoji</td><td>Paste the previously copied emoji sequence frame set to the currently selected position|==CTRL+V==</td></tr>

   <tr><td>Delete Expressions</td><td>Delete the currently specified expression data of the standing avatar|==CTRL+D or DELETE==</td></tr>


</table>


## Parts Management
Used to manage the parts of this standing avatar, you can freely define which parts there are, and each part can be bound to one standing avatar (loop nesting is not allowed).
<table>
  <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
  </tr>
   <tr>
   <td>Select the part</td> <td>Left-click to select the specified part, and edit its properties in the Properties panel after selection</td>
   </tr>

   <tr><td>Adjusting the position</td><td>You can drag and drop or ALT+up and down arrow keys to adjust the position. Position affects the display hierarchy. The lower the layer, the more forward it is displayed</td></tr> 
   <tr><td>Add a new part</td><td>Select a part from the part list to add<br>The list of expressions is managed by number-name unification, and the standing avatar may have multiple parts<br>The meaning of the part within the game depends on the specific item</td></tr>

   <tr><td>Replacement of parts</td><td>Replacement of pre-existing parts</td></tr>

   <tr><td>Remove Part</td><td>Remove selected part</td></tr>

   <tr><td>Jump</td><td>Jump to the specified standing avatar part for editing</td></tr>
   <tr><td>Show / Hide</td><td><img src="img/2/eye.png" /> Show and hide the widget, only for preview use in the editor, hidden widgets cannot be selected in the work area  </td></tr>
   <tr><td>Locking</td><td><img src="img/2/lock.png" />  Lock the part, the locked part cannot be selected in the work area</td></tr>

</table>
