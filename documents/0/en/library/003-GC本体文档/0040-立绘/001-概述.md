# Overview of the standing avatar system
A standing avatar system is an advanced display object that has expressions, parts, and sequential frame maps, and can be considered as a avatar without direction, refer to: [Concept:Avatar](?file=003-GC本体文档/0030-行走图/001-概述)。
> Procedurally speaking standing avatar is a subclass of avatar, and can also be considered as a special avatar.

### Application Scenarios
1.The standing avatar can be displayed in the interface as a avatar component.<br>
2.Used in scripts by the creator on any occasion: as an advanced display object that can be added under any display container at will.
