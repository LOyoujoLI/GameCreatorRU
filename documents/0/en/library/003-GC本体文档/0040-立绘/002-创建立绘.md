# Create a standing avatar


## Enter the standing avatar editor
1-Open the database, click the database icon or shortcut key F10 to open the database<br>
<img src="img/new/3-2.png" /><br>
2-Switch to the standing avatar editor<br>

## Data List
The standing avatar data is managed uniformly by number.<br>
Divided into 16 categories, the maximum value can be changed in each category in order to adjust the range of available standing avatar.<br>
<img src="img/2/立绘/1-1.png" /><br>
<img src="img/2/立绘/1-2.png" /><br>
<img src="img/2/立绘/1-3.png" /><br>


## Select the data for the standing avatar
Left mouse click on the specified standing avatar to open the standing avatar data for editing.
