# Embedded script calls
In addition to writing the core code of the project in the script editor, there are a number of places in the visual editor where code writing can be used.<br>


## Event commands: client-side scripts, server-side scripts
Execution of the script starts when the line of command is executed during execution of the event. <br> You can access the following local variables in addition to the project related code when executing this script. <br> Support for [Local trigger variables](#Local trigger variables)
> For example, if a line of code is executed: trace(123), then when the command is called, 123 will be displayed on the console
