## Using the Script Editor in the Visual Editor
### Start or close the script editor
Use F11 to open or close the script editor, or click the script button above.<br>
<img src="img/2/脚本/1.png" /><br>

### Project and library presets for standalone version
The script editor contains a Game program project by default and it references a GameCreatorLib client-side GC game engine library.<br>
The default game launch starts this Game program item.<br>
<img src="img/2/脚本/2.png" /><br>

### Web Edition Preset Projects and Libraries
The network version provides the Game program project used by the client by default, GameCreatorLib client GC game engine library.<br>
as well as providing the Server Server program project and the GameCreatorServerLib server-side GC game engine library for server-side use.<br>
It also provides a Common library, which can write some common algorithms or logic used in the client and server in one place.<br>
<img src="img/2/脚本/3.png" /><br>

## About the GameCreator Game Engine Library
The GameCretor engine library contains the client-side engine library (GameCreatorLib) and the server-side engine library (GameCreatorServerLib). <br> 
The single-player game template contains only the client-side engine library, while the MMO template contains both (server uses the server-side engine library and client uses the client-side engine library). <br> 
The client-side engine library is the library that encapsulates functional modules such as scene, scene object, avatar, interface, animation, etc. <br> 
The server-side engine library is the library that encapsulates functional modules such as network communication (including RPC), multi-threaded scene management, players, database, etc. <br>
> The visual game editor and game client of GC template are developed based on GameCreator client-side engine library, and the server-side is developed based on server-side engine library.

==Basically every game template will contain the engine library, while the project level code of the template is not the same.==
