Scenario-related triggers may exist in the scenario.<br>
For example, the "Feather of cyan" template has a trigger event called "Enter Scene".<br>
Advanced developers can set custom trigger categories, please refer to [Custom Trigger Event Categories](?file=003-GC本体文档/1010-自定义编辑器/025-自定义触发事件类别)。
