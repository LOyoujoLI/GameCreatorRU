# Set the whole picture as a map

</style>
Use a whole image to display as a map.
<img src="img/new/2-13.png" />


## principle
use [Picture layers](?file=003-GC本体文档/0010-场景/003-图层#新建图片层) Perform the whole picture setup.

## Operation steps
  1.Right mouse click on the layer and select New Layer - Picture Layer<br>
  <img src="img/new/2-14.png" /><br><br>
  2.Click the Select Image button in the Layer Properties panel<br>
  <img src="img/new/2_15.jpg" /><br><br>
  3.Select a picture（[How to import your own resources](?file=003-GC本体文档/0008-资源/002-导入资源)）<br>
  <img src="img/new/2-16.png" /><br><br>
  4.Allows editing the image directly on the scene in edit mode of the layer, controlling its displacement, rotation, scaling, and editing other layer properties in non-editing mode<br>
  <img src="img/new/2-17.png" /><br><br>

## Multiple images displayed simultaneously
  Repeat the above steps to add multiple image layers and overlay them.<br>
  When combined with the Far View property setting, you can also create a multi-layered far view effect.<br>
