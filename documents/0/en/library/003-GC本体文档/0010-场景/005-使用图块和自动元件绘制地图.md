# Drawing maps with blocks and automatic components
<style>
.g{
  color:#FF0000;
  cursor:pointer;
  font-size:22px;
}
</style>
The use of blocks and automatic components allows you to save material and reuse the same elements for multiple drawings in multiple places: for example, houses, trees, rocks, etc.<br>
The size of each block grid is determined uniformly according to the editor's preset 「grid pixel size」 (Menu-Settings-Scenes module-Grid pixel size).<br>
The block/auto component image material is not limited in width and height (but please keep it below 4096x4096 if it is to be mobile compatible).<br>
You can freely use any block material or automatic component material to draw on the same map.

## Importing block material
1.Switch to the ==Block tab bar== in the lower left area of the editor, then click on the menu in the red box<br>
<img src="img/new/2-18.png" /><br><br>
2.Select New Block. (You can also create a new block by right-clicking on an existing block)<br>
<img src="img/2/场景/5-3.png" /><br><br>
3.Select a block image（[How to import your own resources](?file=003-GC本体文档/0008-资源/002-导入资源)）<br>
<img src="img/new/2-19.png" /><br><br>
4.Finish<br>

## Replace block material
1.Right-click on an existing block material and select-<u>Replace block material</u> <br>
<img src="img/2/场景/5-6.png" /><br><br>

<img src="img/2/场景/5-4.png" /><br><br>

<img src="img/2/场景/5-5.png" /><br><br>

## Importing automatic components
Automatic components are special block materials that determine which specific block to display based on the surrounding automatic components that have been drawn<br>
1.Switch to the ==Automatic Components tab bar== in the lower left area of the editor and click on the menu in the red box<br>
<img src="img/2/场景/5-6.png" /><br><br>
2.Select New Auto Component (right mouse click on an existing auto component can also create a new auto component)<br>
<img src="img/2/场景/5-7.png" /><br><br>
3.Select an automatic component picture（[How to import your own resources](?file=003-GC本体文档/0008-资源/002-导入资源)）<br>
<img src="img/2/场景/5-8.png" /><br><br>
4.Select specifications (currently only GCAT1 is supported, size 3x5 half frames, if a frame is 48 pixels then the size should be 72x120) for specific instructions refer to the following:<br>
<img src="img/2/场景/5-9.png" /><br><br>
5.Finish<br>
<img src="img/2/场景/5-10.png" /><br><br>

## Set block grid data
The grid data of the block is pre-set and will be available when it is painted on the map (the specific functions are implemented by the project layer and may vary from project to project).<br>
The setup requires switching to the specified grid data layer label in the block source, such as the pre-defined barrier layer in the "RPG - Feather of cyan" template.<br>
<img src="img/2/场景/5-13.png" /><br>
<table>
 <tr>
    <td><b>Operation</b></td>  <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Open</td>  
    <td>
      ==Single block painting==: Left mouse button click on the block source or press to start painting<br>
      ==Area Painting==: Right mouse button to select an area for painting
    </td>
 </tr>
 <tr>
    <td>Close</td>
    <td>
      == Single block cancellation settings==: SHIFT+left mouse button click on the block source or press and start painting style cancellation settings<br>
      ==Area cancellation settings==: SHIFT+right mouse button to frame an area for paint-style cancellation settings
    </td>
 </tr>

 </table>
