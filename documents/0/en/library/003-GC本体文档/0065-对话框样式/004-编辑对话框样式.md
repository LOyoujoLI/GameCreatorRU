## Preview Comic Dialog Mode
Comic dialog mode means that the dialog appears at coordinates that are no longer relative to the screen, but relative to the specified scene object. <br> Check Preview Comic mode to bring up the reference behavior map, and adjust the dialog box components to fit the avatar. <br>
==This is for preview reference only, the actual displayed object avatar is based on what is set in the [Show Dialogue Event Command](?file=003-GC Ontology Document/0090-Event System/010-System Command-Display Dialogue) executed during in-game runtime.==

<img src="img/2/对话框/1-4.png" /><br>

## Application Scenario
Select the dialog style when calling the event command [Show Dialog Event Command](?file=003-GC Ontology Document/0090-Event System/010-System Command - Show Dialog).<br>
If you want to use the comic dialog, you need to specify the scene object in the options of the comic dialog mode.<br>
