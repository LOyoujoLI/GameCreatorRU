# Concept: dialog box style
Dialog styles are used for dialog events and options, and one style must be used for each dialog sentence. <br> 
Any number of dialog styles are supported within a game, freely set by the author. <br> 
Support for comic dialog mode (dialogs are no longer relative to the screen but bound to scene objects).
