# Creating Dialog Styles


## Go to the dialog style editor
1-To open the database, click the database icon or shortcut key F10 to open the database<br>
<img src="img/new/3-2.png" /><br><br>
2-Switch to the dialog style editor<br>

## Data List
Dialog style data is managed uniformly using numbering. <br> 
Divided into 16 categories, the maximum value can be changed in each category in order to adjust the range of available dialog styles. <br>
<img src="img/2/对话框/1-.png" /><br>

## Select dialog style data
Left-click the specified dialog style to open the dialog style data for editing.
