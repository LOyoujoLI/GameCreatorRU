# Object switch settings
There will be object switches in games with scene objects. An object switch is a switch that is bound to the object, similar to a switch used locally.<br>
For example, object switch A of some NPC1 is on, while object switch A of NPC2 is off.<br>
The object switch will not be reset after switching scenes and reading the file.

### Selecting scene objects
Select an object to operate the object switch on it. <br> 
Trigger object: the object that triggers the event. Who the specific trigger is is determined based on the trigger setting of the specific event. <br> 
Current executor: the object that executes the event, who the specific executor is is determined based on the executor settings of the specific event. <br> 
Assigned object: the object that assigns a scene object based on the number of the current scene.

### Switch number
The system gives 7 switches by default, namely ABCDEFG. You can set more switches using the script.
