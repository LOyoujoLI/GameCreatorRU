# Scene vibration
Use this command to make the scene vibrate for a period of time. <br>
<img src="img/2/事件/7-8.png" /><br>
## Operating Instructions
<table>
 <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Adjust the intensity</td> <td>Select the intensity of the vibration, 5 means 5 pixel random jitter</td>
 </tr>
 <tr>
    <td>Duration</td>
    <td>
     Continuous game frame rate, 60 fps is about 1 second
    </td>
 </tr>
 <tr>
    <td>Preview effect</td>
    <td>
      Click the Preview Effect button to preview the vibration effect
    </td>
 </tr>
 <tr>
    <td>Zoom preview area</td>
    <td>
    Mouse wheel to zoom, shortcut key Z to restore to 100%
    </td>
 </tr>
 <tr>
    <td>Mobile Preview Area</td>
    <td>
      Left mouse button or right mouse button to drag and drop the scene
    </td>
 </tr>
 </table>
