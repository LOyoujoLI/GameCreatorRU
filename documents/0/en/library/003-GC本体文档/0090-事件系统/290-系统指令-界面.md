# Interface
Interface-related events are used to display the interface, close the interface, and modify the interface.<br>


## Display Interface
Select a screen to display, only one screen can be open at the same time using this command.<br>
This interface belongs to the system group interface, i.e. it is managed by using the number of the interface to display, close and operate it by the number of the interface.<br>
Corresponding script command: GameUI.show(1); 1 means interface #1.<br>

## Close the interface
Select an interface to close. Using this command will close the interface of that number in the opened system group. <br> Corresponding script command: GameUI.hide(1); 1 means interface number 1.

## Mobile interface components
The Move Interface Component command allows you to change the properties of components in an interface, such as position, width and height, transparency, image address, and other properties. <br> 
By selecting the specified interface and directly modifying the properties of the component, the system will list the modified properties based on the current modification compared to the preset values. <br>
