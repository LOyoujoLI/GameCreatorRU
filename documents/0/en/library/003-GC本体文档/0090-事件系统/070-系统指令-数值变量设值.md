# Numeric variable setting
Standalone: A type of variable tied to the archive record, with a separate set of variables for each archive (including numeric, switch, and string variables). <br> 
Network version: a type of variable tied to a player, with a separate set of variables for each player (including numeric, switch, and string variables). <br> 
Numeric variables are a type of variable that records numbers and supports recording decimals.

    <td>Assignment method</td>
    <td>
      =: means the variable records the value directly. For example, if "my good and evil value" = 100, it means "my good and evil value" is currently recorded as 100<br> 
      +=: means the variable adds a new value to the current one. If "My Good and Evil Value" +=100, it means "My Good and Evil Value" = "My Good and Evil Value" +100<br> 
      -=: It means the variable is subtracted from the current value. For example, if "my good and evil value" -=100, it means "my good and evil value" = "my good and evil value" -100<br> 
      \*=: It means the variable is multiplied by the new value from the current one. For example, \*=100 means "my good and evil values" = "my good and evil values"\*100<br> 
      /=: means the variable is divided by the new value on top of the current one. For example, "my good and evil value"/=100 means "my good and evil value"="my good and evil value"/100<br>
    </td>
 </tr>
 <tr>
    <td>Value Selection</td>
    <td>
      Fixed value: the value is a fixed value<br> 
      Global/diurnal value: the value is the value recorded by the specified global/diurnal variable<br> 
      Numeric variable: the value is the value recorded by the specified numeric variable<br> 
      Random number: the value is a random value within the specified range, 0~100 means one of the 101 numbers including 0 and 100 is obtained randomly<br> 
      Common function: the value is the value of the custom common <br> 
      Game value: The value is based on the preset value of the custom game value, which is displayed in real time, and different items may have different game values. Such as game gold<br>
    </td>
 </tr>
