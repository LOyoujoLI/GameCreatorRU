# Change the scene tone
Use this command to change the scene [hue](?file=003-GC本体文档/5000-附录/031-色调)。<br>
<img src="img/2/事件/7-9.png" /><br>
## Operating Instructions
<table>
 <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Adjusting tones</td> <td>Reference to [[Tone](?file=003-GC本体文档/5000-附录/031-色调)] for tones description</td>
 </tr>
 <tr>
    <td>Transition time</td>
    <td>
     From the current frame rate, 60 frames is about 1 second. When this command is executed, 1 frame will be executed immediately, if it is set to 1 frame, it means the change will be made immediately.
    </td>
 </tr>
 <tr>
    <td>Zoom preview area</td>
    <td>
    Mouse wheel to zoom, shortcut key Z to restore to 100%
    </td>
 </tr>
 <tr>
    <td>Mobile Preview Area</td>
    <td>
      Left mouse button or right mouse button to drag and drop the scene
    </td>
 </tr>
