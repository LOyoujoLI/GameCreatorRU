# Event Page Snippets
An event page fragment is used to partially execute an event page instruction. <br> It blocks the execution of the outside event page, and the outside event page can continue to execute only after the event page fragment has finished executing.

## Standalone event page snippet
Creating a separate trigger to execute the fragment event does not block the execution of the current trigger. <br> 
The trigger and executor of the newly generated standalone trigger are the same as the currently invoked event page. <br> 
If the trigger and executor of the standalone fragment page event are both the player's scene objects when crossing scenes, the trigger is not destroyed.
