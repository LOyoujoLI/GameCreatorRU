# Event Library
An event library is a set of pre-defined event pages that allow event pages everywhere to call these events <br> 
for reusability.

## Creating an Event Library


### Go to the Event Library Editor
1-Open the event library, click the event library icon or shortcut key F4 to open the event library<br>
<img src="img/2/行走图/1-1.png" /><br>

### Select public event data
Left mouse click on the specified public event to open the public event data for editing
