# Global or 2-week switch settings
Standalone - Second Week Switch: A switch not related to archiving, which is loaded when starting the program. This switch will be included in the call to save global data for saving. <br> 
Network version - Global switch: A common switch for the whole game (service), which can be understood as a game world switch. <br> 
A switch variable that records both on (ON/1) and off (OFF/0) states.
