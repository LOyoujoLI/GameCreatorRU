# Switch settings
Standalone: A type of variable tied to the archive record, with a separate set of variables for each archive (including numeric, switch, and string variables). <br> 
Network version: a type of variable tied to the player, with a separate set of variables for each player (including numeric, switch, and string variables). <br> 
One type of switch variable that records both on (ON/1) and off (OFF/0) states.

## Switch setting panel
<img src="img/2/事件/8.png" /><br>
Select a switch for recording the status and set it to ON or OFF.
