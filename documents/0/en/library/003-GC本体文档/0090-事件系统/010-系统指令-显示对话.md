# System command: Show dialogue
Dialogue is a common form of presentation within the game, and this command allows the screen to display the text of the dialogue being played.

## Dynamic variable symbols
Dynamic symbols are used to display some variable information, not as fixed characters, but as determined by the actual current value at runtime.
<table>
 <tr>
    <td><b>Dynamic variable symbols</b></td>  <td><b>Description</b></td>

 <tr>

 <tr>

      ==[$s1]== 1 means string variable number 1

 <tr>

 <tr>

    </td>

## Waiting for events submitted by players
The appearance of this event suspends the execution of subsequent event commands until the player submits them and then continues.

## Blocking events
This event is a blocking event. Blocking events are only allowed one at a time within the game, and if multiple triggers have blocking events, they will take turns waiting for the player to submit them.<br> 
Blocking events must be [Waiting for events submitted by players](#Waiting for events submitted by players).
