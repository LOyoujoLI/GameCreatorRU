# Variables
Variables in the event system are a recording tool used for logging and are managed uniformly using number-name. <br> Once recorded, the variable can be used everywhere, e.g., the ==5== numeric variable with the name ==Plot Progress== and the current value of ==15==. <br><br>

## Object Switch
In games with scene objects, there will be object switches. An object switch is a switch that is bound to that object, similar to a switch used locally. <br> 
For example, object switch A is on for a certain NPC1, and object switch A is off for NPC2. <br> 
Object switches are not reset after switching scenes and reading files.


## Binding variable display in the interface
[Numeric variable component-UIVariable](?file=003-GC本体文档/0070-界面/008-变量组件UIVariable) shows the value of the specified numeric variable can be displayed in real time. <br> 
[Image component-UIBitmap](?file=003-GC本体文档/0070-界面/006-图片组件UIBitmap) shows the address of the specified string variable in real time. <br> 
[Text component-UIString](?file=003-GC本体文档/0070-界面/007-文本组件UIString) shows the value of the specified string variable in real time. <br> 
[Switch Component-UISwitch](?file=003-GC本体文档/0070-界面/016-开关组件UISwitch) shows the state (on or off) of the specified switch variable can be displayed in real time. <br>
