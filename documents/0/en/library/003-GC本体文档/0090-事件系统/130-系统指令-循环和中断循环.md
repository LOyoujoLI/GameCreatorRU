# Loops and interrupt loops
When this instruction is encountered, it begins to repeat its internal instructions unless an interrupt loop or interrupt command is encountered.

## Tips
If you want to render certain effects gradually, you can wait 1 frame internally. For example, change the rotation of the lens each frame. <br> If you don't wait, it will be done instantly, i.e. you won't see the rotation anymore. <br>
