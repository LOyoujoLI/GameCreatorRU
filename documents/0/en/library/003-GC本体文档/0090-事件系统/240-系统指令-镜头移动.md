# Camera movement
Use this command to change the shots on the scene.<br>
<img src="img/2/事件/7-10.png" /><br>
## Operating Instructions
<table>
 <tr>
    <td><b>Operation</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Use coordinates</td> <td>Lock the camera position to the specified coordinates<br>Quick setup: mouse click on the open space in the preview area to bring up the menu, select <u>Lens Point Here</u></td>
 </tr>
 <tr>
    <td>Specified objects</td>
    <td>
     Specify an object on the scene to lock the camera to<br> 
     Quick Setup: mouse click on an empty space in the preview area to bring up a menu and select <u>Camera Point to Object</u> <hr> 
     Player Object: the player itself, i.e. the object in the birth point<br> 
     Trigger Object: the object that triggers an event, who the trigger is is determined by the trigger settings for the specific event<br> 
     Current executor: the object that executes the event, who the specific executor is is determined by the executor setting of the specific event<br> 
     Designated object: a scene object is designated based on the number of the current scene
    </td>
 </tr>
 <tr>
    <td>Jogging and time frames</td>
    <td>
     Move to the specified camera position within the set time frame. 1 frame will be executed immediately when this command is executed; if set to 1 frame, it means immediate change<br> 
     Jogging: move to the shot in a jogging rather than uniform motion (fast and then slow)
    </td>
 </tr>
 <tr>
    <td>Zoom preview area</td>
    <td>
    Mouse wheel to zoom, shortcut key Z to restore to 100%
    </td>
 </tr>
 <tr>
    <td>Mobile Preview Area</td>
    <td>
      Left mouse button or right mouse button to drag and drop the scene
    </td>
 </tr>
 </table>
