     <td>Frame</td>  <td>
       The frame indicates that the screen/image is displayed once, regarding the frames within GC:<br> 
       The number of frames to wait for in the event command is about 1/60th of a second per frame, which fluctuates depending on the actual game frame rate, for example, if the game is very stuck and drops to 30 frames, then the wait for 60 frames is about 2 seconds<br> 
       The frame rate of the avatar/standing avatar/animation: allows the user to set it by himself and decide the number of frames to play per second, for example, 20 frames per second, that is Each frame is about 1/20th of a second
