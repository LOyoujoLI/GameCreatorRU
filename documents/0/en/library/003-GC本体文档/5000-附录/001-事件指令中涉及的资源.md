# Resources involved in the event command

 The system commands are listed here and are implemented by the engine layer. <br>
 And custom commands are implemented by the project layer itself (by overriding AssetManager.preLoadCommandPage) depending on the project.
 <table>
  <tr>
     <td><b>System commands</b></td> <td><b>Description</b></td>
  </tr>
  <tr>
     <td>Show Dialog</td> <td> 
      Preload to show dialog styles, avatars, and voices involved in the dialog
     </td>
  </tr>
  <tr>
     <td>Display Options</td> <td> 
      Preload to show the style of the dialog involved in the conversation
     </td>
  </tr>
  <tr>
     <td>Object Behavior</td> <td> 
      The avatar resource involved in "Change avatar" contains all the faces, actions, and parts involved within that avatar 
     </td>
  </tr>
  <tr>
     <td>Show fog</td> <td> 
      Show image resources involved in fog. 
     </td>
  </tr>
  <tr>
     <td>Audio</td> <td> 
      Audio resources involved in "Play BGM" "Play BGS" "Play SE" "Play Voice" 
     </td>
  </tr>
  <tr>
     <td>Callout Interface</td> <td> 
      Specify all resources within the interface that needs to be called out
     </td>
  </tr>
  <tr>
     <td>Public Events</td> <td> 
      Related resources covered in Public Events, same as this appendix 
     </td>
  </tr>
  <tr>
     <td>Event Snippets</td> <td> 
      Related resources involved in event snippets, same as this appendix
      </td>
  </tr>
  </table>
