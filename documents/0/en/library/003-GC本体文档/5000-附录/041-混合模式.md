# Appendix V: Blend Mode
Indicates the source image (the image with blending mode set) and the target image (the part of the image that has been rendered). <br> 
Normal: overlays the bottom image display. <br> 
Additive: indicates that the pixel color points of the image will be added together to display the image when it is displayed instead of overwriting it. <br> 
For example, a black #000000 pixel point in the source image means that R,G,B are all 0. Any target pixel point and 0 are still added to get the original color of the target pixel point, which is represented by removing the black background of the source image.
