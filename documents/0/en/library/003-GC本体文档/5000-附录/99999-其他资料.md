# Appendix: Additional Information

## GC Cache
GC can be regarded as a browser, browsing files and so on will keep some cache (such as watching B site videos in GC and so on), can be cleared, the specific location is as follows:<br>
<b>C:\Users\{UserName}\AppData\Local\gamecreator{X}</b><br>
Where UserName indicates the user name of your computer and X indicates a number, usually 1.

## Open History Project
GC Software's history file address:<br>
<b>Software root directory/kdsrpg/workspace.history</b><br>
If it needs to be cleared, it can be set to []。


## Relative Path
Relative paths generally refer to paths relative to the project directory, after packaging and releasing the game due to uncertainty about where the user installed it,<br> generally in-game resource calls are made using relative paths, such as asset/image/xxx.png<br>
==Note: Use the slash '/', not the backslash '\\'.==
