# Appendix: Nine-Gauge
Setting the nine-box property of an image allows the image to stretch in a nine-box fashion when stretched <br> 
so that the edges are not blurred when zoomed in. <br>
<img src="img/2/附录/1-1.png" />

<table>
 <tr>
    <td><b>Nine grid stretching method</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>①-Top left corner</td> <td>Not stretched when changing image size</td>
 </tr>
 <tr>
    <td>②-Top</td>
    <td>
      Stretch proportionally when changing the width of the image, and unchanged when changing the height
    </td>
 </tr>
 <tr>
    <td>③-Top right corner</td>
    <td>
    No stretching when changing image size
    </td>
 </tr>
 <tr>
   <td>④-Left</td>
   <td>
   No change when changing the width of the image, stretch proportionally when changing the height
   </td>
 </tr>
 <tr>
   <td>⑤-Bottom left corner</td>
   <td>
   No stretching when changing image size
   </td>
 </tr>
 <tr>
   <td>⑥-Bottom</td>
   <td>
   Stretch proportionally when changing the width of the image, and unchanged when changing the height
   </td>
 </tr>
 <tr>
   <td>⑦-Bottom right</td>
   <td>
   No stretching when changing image size
   </td>
 </tr>
 <tr>
   <td>⑧-Right</td>
   <td>
   No change when changing the width of the image, stretch proportionally when changing the height
   </td>
 </tr>
 <tr>
   <td>⑨-Middle</td>
   <td>
   Stretch proportionally when changing image width and height
   </td>
 </tr>
 <tr>
   <td>flattening</td>
   <td>
   Change from stretch to tile when the original size of the image is exceeded
   </td>
 </tr>
 </table>

 <tr>

    <td>Adjusting the nine-pane grid</td>
    <td>

    </td>
 </tr>
