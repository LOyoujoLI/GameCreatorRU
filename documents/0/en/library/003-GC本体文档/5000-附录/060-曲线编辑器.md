# Appendix: Curve Editor
Use the curve editor to edit a set of relationship data, such as for level-experience requirement relationships or level-attribute growth relationships. <br> With an upper limit referenced to the data maximum, the curve is calculated from left to right to obtain the current value based on the curve between two nodes. <br>

 <tr>

 <tr>
    <td>Curve editing: select nodes</td> <td>Left mouse click on nodes</td>
 </tr>
 <tr>
    <td>Curve Edit: Create Node</td> <td>Double-click the left mouse button to create a node</td>
 </tr>
 <tr>
    <td>Curve editing: setting the node transition method</td>
    <td>
       Right mouse button to select linear node or secondary Bezier curve node<br> 
       Linear node: adjust the two nodes position according to the position of the two nodes to transition evenly<br> 
       Secondary Bezier curve node: adjust the curve according to the position of the two nodes and the red control point to transition according to the way of the curve<br>
    </td>
 </tr>
 <tr>
    <td>Curve Edit: Delete Node</td>
    <td>
       Use the DELETE key or the right mouse button menu to delete the currently selected node<br> There will be at least two nodes, the starting point and the ending point, the starting point must be at 0% and the ending point at 100%
    </td>
 </tr>
 <tr>
    <td>Segment Preview</td>
    <td>
       Data values for simulated segments only are displayed to aid in calculations
    </td>
 </tr>
 <tr>
    <td>Preset configuration</td>
    <td>
       You can save the current curve settings as a preset configuration or switch to a preset configuration<br> 
       Add a preset: Left mouse click on the + sign to add a new current curve data as a preset<br> 
       Apply a preset: Left mouse click on an existing preset to switch to that preset configuration<br> 
       Edit a preset: Right mouse click on an existing preset to rename it, delete it, replace it with the current setting, or Modify prompt text
    </td>
 </tr>
