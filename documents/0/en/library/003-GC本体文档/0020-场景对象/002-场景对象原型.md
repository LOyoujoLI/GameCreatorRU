# Scene object prototype
A scene object prototype is a predefined class of objects consisting of ==public modules== and ==pre-installed scene object modules==.<br>
Each scene object has its prototype, and the prototype has the characteristics that the object has. (For example, Xiao Ming's prototype is a human, while Wang Cai's prototype is a dog)<br>
Advanced developers can edit the prototype, please refer to the detailed information [Custom Editor - Scene Object Prototype](?file=003-GC本体文档/1010-自定义编辑器/020-场景对象原型)。


>For example, two prototypes, human and animal, are defined, and they both possess the property of movement speed. We implement the properties common to all species in a common module<br>
