# Scene Object Module
Scene object modules are assemblable components. When a scene object has a module, it has the characteristics of that module.<br>
Contains ==properties==, ==display objects==, ==binding scripts==, ==editor preview==<br>
Advanced developers can make their own object modules, for details please refer to [Custom Editor - Scene Object Module](?file=003-GC本体文档/1010-自定义编辑器/022-场景对象模块)。<br>

> For example, if an object has a shadow module attached to it, then the object has a shadow and the shadow style can be previewed in real time by adjusting the shadow parameters.
