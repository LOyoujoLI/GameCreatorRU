## Placement object
Way 1: Drag and drop the preset object to the specified location in the scene, if == hold down the SHIFT key will be aligned with the grid center point ==<br>
<img src="img/new/2-32.png" /><br><br>
Way 2: Right click on the empty space - <u>New Scene Object</u> (must have switched to the object layer)<br>


## Adjusting object position
Way one: directly drag and drop the scene object to the specified location, == hold down the SHIFT key will be aligned with the grid center point ==<br>
<img src="img/new/2-33.png" /><br><br>
Way two: use the keyboard arrow keys to adjust the position of the currently selected scene object (hold down SHIFT + arrow keys then move one square at a time)<br><br>
Way three: adjust the position of the coordinates in the right property bar to change the coordinates (you can press and hold the text and drag)<br>
<img src="img/new/2-34.png" /><br><br>

## List of scene objects
 Click on the Scene Object List button in the upper left corner of the scene (or ==TAB key==) to call out the Scene Object List:<br>
 <img src="img/2/场景对象/1-16.png" /><br>
 After calling out the list, you can double-click to select the scene object, or use the right-click menu function.<br>
 <img src="img/2/场景对象/1-17.png" />
 <img src="img/2/场景对象/1-18.png" />
 <img src="img/2/场景对象/1-19.png" /><br>

## Positioning
 For the selected scene object you can press F to position it so that the current editor's scene lens is pointed at it.<br>
 The scene object can also be located in the list of scene objects by ==double-clicking the left mouse button== or ==right-click menu== in the ==location== of the scene object.<br>

## Processing of overlays
 The current mouse position of the superimposed characters will be listed in the menu after the right mouse button is clicked, and you can switch to select them.<br>
 It is also possible to select from the list of scene objects.<br>
