# Scene object triggers
Support binding custom triggers, such as click on the object's event, touch the object's event, etc., customize a trigger category name and implement the trigger in the script.<br>
Multiple events can be executed simultaneously if the execution conditions are met.<br>
Advanced developers can customize the trigger categories, please refer to [Custom trigger event categories](?file=003-GC本体文档/1010-自定义编辑器/025-自定义触发事件类别)。<br>
