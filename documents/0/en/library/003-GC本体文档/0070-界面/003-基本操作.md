## Alignment Tool
Multiple selected components will align the parent container with their enclosing box calculations<br>
<table>
 <tr>
    <td><b>Functions</b></td> <td><b>Description</b></td>
 </tr>
 <tr>
    <td>Horizontal left alignment</td> <td>Selected single or multiple components align to the left border of the parent container</td>
 </tr>
 <tr>
    <td>Horizontal centering</td> <td>Selected single or multiple components aligned to the horizontal middle (at 1/2 the width) of the parent container</td>
 </tr>
 <tr>
    <td>Horizontal righting</td> <td>Selected single or multiple components align to the right border of the parent container</td>
 </tr>
 <tr>
    <td>Top vertically</td> <td>Selected single or multiple components align to the top boundary of the parent container</td>
 </tr>
 <tr>
    <td>Vertical centering</td> <td>Selected single or multiple components aligned to the vertical middle (at 1/2 the height) of the parent container</td>
 </tr>
 <tr>
    <td>Vertical Down</td> <td>Selected single or multiple components align to the lower boundary of the parent container</td>
 </tr>
 </table>

## Component Toolbar
A list of all components that can be created. Drag and drop to the workspace or interface component list to create components. <br> 
When dragging to the interface component list, you can select the component on the same level as the component or drag it inside the component as its sub-level, depending on whether you drag to the left of the component. <br>

## Interface Binding Class
Create an interface binding class that inherits from the system generated class for that interface (e.g. if that interface is interface #1, then it inherits from GUI_1) and can be accessed directly to control that component using ==this.{ComponentName}==.<br>
When the interface is created at game runtime, the binding class for the interface is automatically created as an instance<br>
[Script Editor Documentation Reference](?file=003-GC本体文档/1000-脚本系统/001-概述)
>&nbsp;class GUI_XXX extends GUI_1 {<br>
>&nbsp;&nbsp;&nbsp;&nbsp;constructor() {<br>
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;super();<br>
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;this.MyButton.x = 100;<br>
>&nbsp;&nbsp;&nbsp;&nbsp;}<br>
>&nbsp;}
