# Create Interface


## Enter the interface editor
1-Open the database, click the database icon or shortcut key F10 to open the database<br>
<img src="img/new/3-2.png" /><br><br>
2-Switch to interface editor<br>

## Data List
The interface data is managed uniformly using numbering.<br>
Divided into 16 categories, the maximum value can be changed in each category in order to adjust the range of interfaces available.<br>
<img src="img/2/界面/1-1.png" /><br>

## Select interface data
Left mouse click on the specified screen to open the data of that screen for editing.
