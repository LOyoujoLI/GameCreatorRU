# Plugin Overview
Plug-ins are additional components that are pre-made features or resources by the author that can be easily provided to other users game projects to install or uninstall. The plug-in system has the following features.<br>
<li>Install and uninstall at any time</li> 
<li>Extend the functionality and modules of the editor</li> 
<li>Extend the functional effects of the game runtime</li> 
<li>Get more pre-defined game resources</li> 
<li>Any user can make plugins and share them for sale</li>
