# About GameCreator
<img src="http://material.gamecreator.com.cn/GC%E5%9B%BE%E6%A0%87%28%E7%B2%BE%29.png" width = "200" height = "200"/>

#### GameCreator is a visual game creation software that is compatible with == hobbyist game creation== and == professional level game creation==.<br>
#### The main characteristics are as follows:<br>
## Different solutions according to different levels of developers<br>
>  -- Visual editors such as scene editor, animation editor and interface editor are provided for designers to edit common game resources for in-game use.<br>
  -- Professional-level code editor (IDE) and engine and framework for programmers, with support for typescript and javascript languages, as well as VSCode for development.<br>
  -- Visual programming (event editor) for amateur game enthusiasts to visually create game content and logic.<br>

## Game Templates<br>
  Advanced developers can customize game editors and scripts, etc. to create game templates with editors specific to that game to share or sell.

  > For example, RPG game templates have avatar editors, skill editors, item editors, status editors, etc. and core RPG-related gameplay,<br>
   While in the card game template may have a card editor, level editor, etc.<br>These editors can be created visually and scripted to achieve functional effects<br>
<img src="img/t2.jpg" />

## Plugin System<br>
Plug-ins are additional components that are pre-made features or resources by the author that can be easily provided to other users game projects to install or uninstall.<br>
Plug-in systems usually have the following features:<br>
<li>Install and uninstall at any time</li>
<li>Extensible editor features and modules</li>
<li>Extendable game runtime functional effects</li>
<li>Access to more pre-defined game resources</li>
<li>Any user can create plug-ins and share them for sale</li>

## Material Workshop<br>
Material Workshop is where material makers can share or sell game material for free or for a fee, allowing game makers to easily purchase licensed material,<br>
It also allows material producers to earn their fair share of revenue.

## Visual Editor<br>
  The system has some built-in visual editors and also allows users to make their own visual editors.<br>
  The editors built into the system at this stage already include, but are not limited to:<br>
  &nbsp;&nbsp;-- [2D Scene Editor]: Support binding scripts, generally composed of map and scene objects<br>
  &nbsp;&nbsp;-- [2D avatar editor】：support 1~8 directions, multi-action, multi-part, multi-frame number management<br>
  &nbsp;&nbsp;-- [2D standing avatar editor]: support multi-expression action, multi-part, multi-frame number management<br>
  &nbsp;&nbsp;-- [2D animation editor]: Nested images/sub-animations/audio, keyframes, K-frames<br>
  &nbsp;&nbsp;-- [2D Particle Editor]: A mapping and multiple parameters to control the particle form<br>
  &nbsp;&nbsp;-- [Scene Object Editor】：Support users to edit the properties, modules and bound scripts of scene objects<br>
  &nbsp;&nbsp;-- [World and Player Settings Editor]: General settings items, such as system settings, player backpack, etc.<br>
  &nbsp;&nbsp;-- [World and Player Settings Editor]: General settings, such as system settings, player backpack, etc. [Dialog Style Editor]: Support for managing multiple dialog box/option styles, with the ability to freely switch multiple styles within the same game<br>
  &nbsp;&nbsp;-- [Interface Editor】：Support visualization to create interface and write logic and binding script program<br>
  &nbsp;&nbsp;-- [Object Behavior Editor]: Support users to create custom object behavior, commonly used in RPG class template arrangement of the avatars behavior on the scene<br>
  &nbsp;&nbsp;-- [Event Editor】：Support users to create custom event commands and preview the effect in real time without running the game<br>
  &nbsp;&nbsp;-- [Custom Editor]: The GC built-in visual custom editor allows users to create custom editors, such as skill editor, prop editor, etc

## Package Release<br>
  GC Cloud Game Platform: Authors can upload and publish their games to GC Cloud Game Platform with one click, and can update their games gradually in order to get players' support, feedback, messages, etc.<br>
  At the same time, if you buy the game software itself, you can also freely package it on other platforms: PC, mobile, WEB, etc.<br>
  Unless a special cooperation game contract is signed with the official, the copyright belongs to the author by default, and there are no additional expenses such as subsequent split payment system.

## Cloud Platform Support<br>
  In addition to the software itself also provides cloud platform service support, such as cloud archiving, cloud ranking SDK access, etc., so that users can get more service support.<br>
  Cloud platform also provides a template mall, plug-in mall, material workshop, community exchange and other functions



## Others you may be interested in<br>
#### Engine Kernel<br>
  The engine kernel does not have any game template logic and requires programmers to create games or game editors based on the modules and APIs provided by the engine.<br>
  GameCreator currently offers a 2D standalone kernel, and will offer a 2D network kernel, a 3D standalone kernel, and a 3D network kernel in the future.<br>
  >The network version means that during the editing process of the game editor, the network function is supported (for example, after placing an NPC and setting the dialogue, all players can see the NPC and can talk to it),<br>
  Also supports server-side code writing (also supports typescript, javascript and supports shared libraries with the client in order to reuse code)

#### Shared Community<br>
  GameCreator is a kind of community sharing software, and the development will be driven by the community in the mature stage, users can share game templates, scripts, plug-ins, art resources and so on.<br>
  Also users communicate, team up and trade with other makers according to their situation and choice.<br>
  The stronger the community, the richer the scripts, plug-ins and material resources that can be generated, the more it can meet the needs of production and the lower the threshold for production.<br>

#### Ecological Support<br>
  We will support high-quality works in the ecology, providing technical support, financial support, version number processing, distribution support, online Steam and other comprehensive services.
