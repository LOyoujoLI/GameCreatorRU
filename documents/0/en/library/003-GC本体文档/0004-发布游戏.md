# Published Games
Users who have purchased the software proper are free to publish it on any platform, and unless a special cooperation contract is signed with the official, the copyright belongs to the author by default, and there are no additional expenses such as subsequent sharing of the payment system.

## concept

### Supported publishing platforms
Windows<br>
Web (web side)<br>
GC Cloud Gaming Platform<br>
==Android (support coming soon)==<br>
==MacOS (support coming soon)==<br>
==IOS (support coming soon)==<br>
==More platforms will be supported in the future==<br>

### GC Cloud Gaming Platform
Authors can upload and publish games to GC cloud game platform with one click and support to update games gradually in order to get players' support, feedback, messages and so on.<br>
Copyright belongs to the author by default, unless a special cooperation agreement has been signed with the official.

### Encryption
The script currently supports a self-developed program script obfuscation encryption.<br>
==More encryption schemes will be supported in the future.==<br>
==We will provide material encryption solutions in the future.==

### Publish to GC Cloud Platform
Choose to publish to [GC Game Platform] to upload your own made game with one click, and the game project content will be displayed on the website after it meets the standard and passes the management audit, and support to update the game gradually.<br>
