# Code example: Avatar and Standing Avatar
Avatar is an advanced display object that has action, direction, part, and sequence frame maps.<br>
Avatar's origin is the crosshair collimation in the editor。<br>
Standing Avatar is a subclass of Avatar, which can be regarded as a special Avatar without direction<br>
[Concept：Avatar](?file=003-GC本体文档/0030-Avatar/001-概述)<br>
[Concept：Standing Avatar](?file=003-GC本体文档/0040-StandAvatar/001-概述)<br>
[Avatar API：Avatar](?file=006-API-2D单机内核/002-客户端/013-Avatar)<br>
[Standing Avatar API：StandAvatar](?file=006-API-2D单机内核/002-客户端/030-StandAvatar)<br>

## Creating an Avatar
Avatar is generally not created separately, if on the scene is generally created scene object, Avatar exists as a sub-display object of the scene object.<br>
> // Create Avatar #5<br>
> var a = new Avatar();<br>
> a.id = 5;<br>
> a.x = 300;<br>
> a.y = 300;<br>
> // Add to the stage<br>
>stage.addChild(a);<br>
> // Start playing<br>
> a.play();<br>

## Create Standing Avatar
> // Create Standing Avatar #3<br>
> var a = new StandAvatar();<br>
> a.id = 3;<br>
> a.x = 300;<br>
> a.y = 300;<br>
> // Add to the stage<br>
>stage.addChild(a);<br>
> // Start playing<br>
> a.play();<br>

## Listening for load completion events
>var a = new Avatar();<br>
>a.on(EventObject.LOADED,this,()=>{<br>
>&nbsp;&nbsp;&nbsp;&nbsp;// to do<br>
>});<br>
>a.id = 5;<br>

## Modify Avatar/Standing Avatar properties
> // Modify the face: face up (Standing Avatar no direction, no need to modify this property) <br>
> a.orientation = 8;<br>
> // Modify frame rate<br>
> a.fps = 20;<br>
> // Modify the action, based on the action ID<br>
> a.actionID = 3;
> // Skip to frame 5<br>
> a.setCurrentFrame(5,true);

## Avatar parts operation
Parts are managed by number-name unification, similar to the action list. For example 1 - head 2 - body<br>
> // Add a part, ignore if the part already exists <br>
> a.addPartByID(Part number, Avatar or Standing Avatar number)<br>
> // Replace a part <br>
a.changePartByAvatarID(Avatar or Standing Avatar number, part number): boolean
 // Remove a part <br>
a.removePartByPartID(Part Number): boolean


## Avatar component and Standing Avatar component
which is essentially a component that encapsulates Avatar and Standing Avatar.
