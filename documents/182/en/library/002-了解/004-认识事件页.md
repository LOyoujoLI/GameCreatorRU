## Start

The event page is where Command are stored. You can also enable the event preview page mode to facilitate previewing and generating Command.

## Relationship between events and Command

An event is an overview of the Command within the entire event page.

A directive, on the other hand, has a clear point of reference, such as displaying a dialogue directive.

## Standalone events

Imagine an event as a team, while checking independent events starts a separate team, which improves efficiency and prevents individual Command from not being executed (while boosting game performance usage).

## Event page preview mode

Event page preview mode can be very convenient for us to quickly preview and generate Command.

Take the event page of the event library as an example, find the event preview page button and click it to open the event preview page mode.

In fact, in AVG, there is an event page preview mode on the right side of the main interface.
