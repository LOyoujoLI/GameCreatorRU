## Start

Command are very common functions, and many logical relationships can be made through Command.

For example, execute a dialogue, determine whether the player has a certain prop or not, and whether the player has passed the level once.

## Use

Here is an example of displaying a piece of dialogue.

By right-clicking [Event Page](?file=002-了解/004-认识事件页) you can open the command page, and then select the **Show Dialog** command within the command page.
