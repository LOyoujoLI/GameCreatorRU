## Menu

### Interface display

![](img/Understand/001/001.png=600-)

### Description

#### file

![](img/Icon/File.png)

> We can create, switch template projects from here

#### withdraw, restore, save

![](img/Icon/withdraw.png)![](img/Icon/restore.png)![](img/Icon/save.png)

> Project withdrawal, restore, save operations (remember to save more)

#### database

![](img/Icon/database.png)

> Involved in the game interface, animation, parameter settings we can find through the database. **For example, the interface of the game, prop settings**.

#### event database

![](img/Icon/event_database.png)

> The default Commandof the template are stored in the event library, and much of the game's effect logic is implemented through here. **e.g. open interface, enter game**. Authors can also write some generic Command here.

#### image explorer

![](img/Icon/image_explorer.png)

> The game's image material is stored here, and you can add or remove image material here.

#### Music Explorer

[](img/Icon/Music_Explorer.png)

> The game's audio clips are stored here, and you can add or subtract audio clips through here.

#### script

![](img/Icon/script.png)

> Code-related, authors who master programming can develop functional extensions through here.

#### test game

![](img/Icon/test_game.png)

> This button allows you to start a test game. Additional operations: ctrl allows you to test the game in the browser. alt allows you to specify a scene to test. Right mouse button shows QR code, by scanning QR code you can test on mobile.

#### Plugin

![](img/Icon/Plugin.png)

> Through plugin management we can install/uninstall plugins.

#### Settings

![](img/Icon/Settings.png)

> Control the overall game settings, we can import font style, game resolution, block size (RPG category) here.

#### community

![](img/Icon/community.png)

> You can visit the official community page directly.

![](img/Understand/001/002.png=600-)

When **start the game**, it will enter the scene map set by **World Settings/Initial Game Scene**, we can write the logical flow of the game in the scene map.

Click on the top right corner of ![](img/Icon/plus.png) to create a new scene map

## Command, event page, explorer

### Interface display

![](img/Understand/001/007.png=600-)

### Description

Command are the common functions used in making games, and the event page is where the Command are placed. Note: The event page in the main interface is the **event preview page**.

In the main interface, the bottom explorer can generate Command quickly.

#### Bottom Explorer

With the bottom explorer we can quickly drag out the corresponding resources and generate Command.

![](img/Understand/001/008.gif=600-)

#### Modify command parameters

We can modify the command parameters by double-clicking on the command.

![](img/Understand/001/009.gif=600-)

#### Quickly modify command parameters

![](img/Understand/001/010.gif=600-)

We can quickly modify the command parameters by selecting the command and then going to the command parameters page at the bottom of the event page.

**Note:**

**1、Only specific Command are supported**

**2. Need to open the event preview page**
