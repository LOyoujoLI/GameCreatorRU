## Start

The nine-pane grid is a powerful image processing feature that allows images with borders to be made without causing stretching.

## Use

Simply turn on the nine-pane grid in the component properties settings.

The adjustment parameters are different for different materials, but one suggestion is to make the yellow line exclude the border as much as possible.
