## Interface layer

The interface layer places **Show Dialog** and **Show Interface** related content.

In the preview area, the content of the image layer in the interface layer is displayed in dark color.

![](img/In-depth/001/001.png=600-)

Note: The **Show Interface** command can modify the display level, so the interface called by the Show Interface command is not necessarily at the interface level.

![](img/In-depth/001/002.png)

## Image layer

Usually we call images, animations, standing avatars, and Video through Command on the image layer, and for interfaces, we need to modify the layer by displaying the interface command.

The interface on the image layer will be displayed in semi-transparent.

![](img/In-depth/001/003.png=600-)

### Numbering

Image layers are controlled by numbering, and numbering is shared across image layers.

Image layer numbering is similar to variable numbering in that only one number can exist.

Suppose there is an image that you want to have a transition effect, it must be controlled by the numbering of this image.

![](img/In-depth/001/004.png=600-)

![](img/In-depth/001/005.png=600-)

If a command with the same number but different calling resources exists, then the new command will overwrite the old one.

![](img/In-depth/001/006.png)
