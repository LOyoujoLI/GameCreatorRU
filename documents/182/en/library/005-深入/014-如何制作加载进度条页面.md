## Start

With the **slider component** within the interface we can quickly make a progress bar effect.

![](img/In-depth/014/001.png)

## Use

The slider component is affected by the **value** within the component property settings, and we can assign this property value via the **move component** command within the interface.

![](img/In-depth/014/002.gif=600-)

This way the attribute **value** of the slider component is influenced by the variable

![](img/In-depth/014/003.png=600-)

## Test

![](img/In-depth/014/006.gif=600-)

The slider component is mouse-influenced by default, but we don't want the mouse to be able to influence the value of this progress bar, so you can uncheck **Mouse Response** .

![](img/In-depth/014/007.gif=600-)

Uncheck **Mouse Response** and you are done.
