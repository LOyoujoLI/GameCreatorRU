## Start

First analyze the requirements.

Use **event pages** to manage all **closures** of **game interfaces**.

We are familiar with event pages, so we can just write them on the **event library** and call them.

The game interface is definitely closed using the **Close Interface** command.

So how to judge the interface? Once it comes to judging we must react to the fact that we can use **Conditional Branch**.

The overall logic is sorted out: determine whether the specified interface is displayed by Conditional Branch, and if so, close the interface.

## Determine the specified interface

Find the interface options through Conditional Branch, select the interface you want to judge, and then the drop-down box we choose **Front of the line**.

Why choose the front-most display? Because in the game there are multiple game interface at the same time, here I want to close the interface is the most front display, of course, the specific choice or you need to judge according to the actual situation.

## Close command

Next, just write close the specified interface within the Conditional Branch.

In addition to the close interface command I also wrote the interrupt command, why do you write this? Because this event page is expected to manage the closing of all interfaces, and the interrupt command is added here to stop the execution of subsequent Command once the conditions are met, reducing unnecessary calculations.
