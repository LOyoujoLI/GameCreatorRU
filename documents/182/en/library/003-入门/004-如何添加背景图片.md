## Start

For adding resource images, we can create them directly through the explorer at the bottom of the main screen. (That is, the Explorer at the bottom in event preview page mode)

The displayed image is the image layer content, what is image layer? You can check **[Layers in AVG](?file=005-深入/001-AVG中的层级关系)**

![](img/GettingStarted/004/001.gif=600-)

## Numbering

The numbering is the id that controls the current image, and the numbering is shared among the image layers.

We need to control with numbering by operations such as display, move, and eliminate.

There can be only one number that exists and is unique.

![](img/GettingStarted/004/002.png)

## Move Picture

The **Move Picture** command allows the displayed picture to have a transition animation effect, and the moving picture is based on the number.

Note: If the transition effect is followed by a directive, you need to add the wait directive to ensure that the transition can be executed properly.

![](img/GettingStarted/004/003.png)

### Event preview page drag and drop creation

You can quickly create a new **Move Picture** command through the preview window of the event preview page.

Note 1: Do not select the command dragging, if you select the command dragging will become a command to modify the current command coordinates.

![](img/GettingStarted/004/004.gif=600-)

### Event preview page right mouse button creation

The **Move Picture** command can be created quickly by selecting for the command, right mouse button.

![](img/GettingStarted/004/005.gif=600-)

### Create via command page

You can also create **Move Picture** via the command page.

![](img/GettingStarted/004/006.gif=600-)

## Delete Image

**Delete Image** command can just delete the displayed picture, the eliminated picture according to the number.

### Event preview page right mouse button creation

The **Delete Image** command can be created quickly by selecting the right mouse button for the command.

![](img/GettingStarted/004/007.gif=600-)

### Create via command page

You can also create **Delete Image** via the command page.

![](img/GettingStarted/004/008.gif=600-)

## Preview and test

### Preview

We can preview the effect of the command directly in the scene map

![](img/GettingStarted/004/009.png=600-)

### Test

We can test the game directly by clicking on the menu **Game Test (shortcut F9)**.

Note: Since the command is placed in the game scene, you need to click **Start Game** after entering the game.

Note: Please make sure the current game scene is **Initial Game Scene**.
