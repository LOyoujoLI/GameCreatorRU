## Start

Switching game scenes is very easy, only one command is needed.

1: Create a new scene map by clicking +

![](img/GettingStarted/005/001.gif=600-)

2: Next, right click on the event page where you wish to execute the command to insert

Find **Change Scene** in **Logic**

![](img/GettingStarted/005/002.gif=600-)

### Test

We can test the game directly by clicking on the menu **Game Test (shortcut F9)**.

Note: Since the command is placed in the game scene, you need to click **Start Game** after entering the game.

Note: Please make sure that the game scene where the current command is executed is **Initial Game Scene**.
