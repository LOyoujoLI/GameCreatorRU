## Start

The whole tutorial we have been able to make a simple case, finally you will make the game content published!

Of course, before publishing, please also pay attention to the following points.

## Is the initial game scene correct

We need to make sure that the **Database -> World Settings -> Basic Setting** is set correctly and that it is the one you want to start the game with.

![](img/GettingStarted/010/001.png)

## Whether the player can start the game properly

Usually the player does not start the game after opening the game, but enters the title screen. Make sure that the title screen has a button to start the game and Command.

Make sure the title screen has a start button and Command. 

![](img/GettingStarted/010/002.png=600-)

### Command to start the game

Actually the **Start_Button** is just a decoration, what is really executed is the **Start** command.

We can find the command in the command page **System -> Start**.

![](img/GettingStarted/010/003.png)

## Test the game

Be sure to test the game once before you pack it to make sure it's okay.

![](img/GettingStarted/010/004.png)

## Start packaging and publishing

We can quickly package and release our game by using **Publish The Game**.

![](img/GettingStarted/010/005.png)

Currently supports windos, Web package release. Among them, Web needs to build their own corresponding server environment before you can play.

![](img/GettingStarted/010/006.png)
