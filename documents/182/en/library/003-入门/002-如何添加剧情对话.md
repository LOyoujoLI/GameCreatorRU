## Start

1、Right mouse click on the event page to insert to open the command page

![](img/GettingStarted/002/001.gif=600-)

2、Find **Show Dialog** in **Message**(If you can't find it, you can find it by searching![](img/Icon/searching.png))

![](img/GettingStarted/002/002.png=600-)

![](img/GettingStarted/002/003.png=600-)

3、On the left we can set the character name for this conversation, the character avatar

![](img/GettingStarted/002/004.png=600-)

![](img/GettingStarted/002/005.png=600-)

4、 in the content area we can directly enter the dialogue content

![](img/GettingStarted/002/006.png=600-)

![](img/GettingStarted/002/007.png=600-)

5、the top of the content area can be adjusted for the dialogue style, parameters. (The main operation here is to modify the color)

![](img/GettingStarted/002/008.png=600-)

![](img/GettingStarted/002/009.png=600-)

6、we can preview the effect in the game by clicking **Preview**

![](img/GettingStarted/002/010.png=600-)

![](img/GettingStarted/002/011.png=600-)

## Preview and test


### Preview

We can preview the effect of the command directly in the scene map

![](img/GettingStarted/002/012.png=600-)

### Test

We can test the game directly by clicking on the menu **Game Test (shortcut F9)**.

Note: Since the command is placed in the game scene, you need to click **Start Game** after entering the game.

Note: Please make sure the current game scene is **Initial Game Scene**.

![](img/GettingStarted/002/013.png=600-)
