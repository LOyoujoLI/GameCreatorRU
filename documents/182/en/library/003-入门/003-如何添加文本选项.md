## Option setting judgment condition

ou can enable different judgment features by clicking on the option settings.

![](img/GettingStarted/003/006.png)

### Default options


The default position for the focus point to focus on when the option appears.

Note: The default option style is based on **Database -> Dialog Style -> Focus Selected Image**.

![](img/GettingStarted/003/007.png)

![](img/GettingStarted/003/008.png)

### Cancellation occasions


When a cancelled occasion is set, you can close the current option by pressing the keyboard shortcut ESC and trigger the command within the Set cancelled occasion option.

![](img/GettingStarted/003/009.png)

### Occurrence conditions


Occurrence conditions can be used to determine whether the current option is shown or hidden by matching the variable judgment.

About variables and the judgment can be seen in **[Recognizing variables](?file=004-基础/002-认识变量)**, where the judgment is similar to **[Condition divergence](?file=004-基础/004-认识条件分歧)**

![](img/GettingStarted/003/010.png)

![](img/GettingStarted/003/011.png)


## Preview and test

### Preview

We can preview the effect of the command directly in the scene map

![](img/GettingStarted/003/012.gif=600-)

### Test

We can test the game directly by clicking on the menu **Game Test (shortcut F9)**.

Note: Since the command is placed in the game scene, you need to click **Start Game** after entering the game.

Note: Please make sure the current game scene is **Initial Game Scene**.

![](img/GettingStarted/003/013.gif=600-)
