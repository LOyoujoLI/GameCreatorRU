## Start

Right click on the event page where you wish to execute the command to insert

Find the command **Play Music** in **Audio**, etc.

The music file to be called is a file in **Music Resource Manager**

![](img/GettingStarted/008/001.gif=600-)

## Transition effects

Fill in the specified number of frames to add a fade-in effect to the music.

![](img/GettingStarted/008/002.png)
