## Hierarchy

Assume that when the button component is placed below the image component, the button component is a child of the image component, and the image component is the parent of the button component, and the child will be controlled by the parent.

![](img/Optional/005/001.gif=600-)

## Axis point

The default axis point of the component is the top left corner, if you want the axis point to be the center you need to change it in the component properties setting.
