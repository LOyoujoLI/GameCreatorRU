## Start

When we need to display the variable resident in the interface, we can use the corresponding variable component to do so.

## Numeric

The display of numeric values we can achieve with **Numeric Variable Component**, **Game Numeric Component**.

![](img/Basic/009/001.png)

![](img/Basic/009/002.png)

### Use

Just drag the component out and use it, with the right side properties you can change the style and the source to get the value.

![](img/Basic/009/003.png=600-)

![](img/Basic/009/004.png=600-)

## Switches

The display of the switches we can achieve with the **Switch component**.

![](img/Basic/009/005.png)

### Use

Just drag the component out and use it. With the properties on the right side you can change the style and the source to get the value.

![](img/Basic/009/006.png=600-)

## String

The display of the switch we can achieve with **text component**, **game string component**.

![](img/Basic/009/007.png)

![](img/Basic/009/008.png)

### Use

Just drag the component out and use it, with the right side properties you can change the style and the source to get the value.

For the text component, you need to change it to string mode first if you want to use variables.
