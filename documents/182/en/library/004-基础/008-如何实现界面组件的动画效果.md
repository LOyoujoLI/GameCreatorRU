## Start

While the previous tutorials talked about the animation of the interface itself, in this tutorial we create the transition of the component by using the **Move Component** command.

Note: To use the **Moving Components** Command, you need to make sure that the specified interface is already displayed, i.e. the **Show Interface** Command.

## Caution

Since this tutorial only has a transition effect, there are no other Command after the transition so there is no need to write **wait**.

Suppose this event page needs to add other Command after the transition, then you need to add wait, otherwise the transition effect will start executing the subsequent Command before it is finished.
