## Start

Conditional Branch is similar to the **Show Choices** command we learned earlier, in that it also executes different Command on a case by case basis.

For example, a dialogue is executed when the player has a certain item, and another dialogue is executed when he does not have a certain item.

### Judgment method

Conditional Branch is executed by satisfying the judgment condition and not satisfying the judgment condition.

By default,Conditional Branch is only executed by satisfying the condition, only if **Create Else Branch** is checked will the judgment of not satisfying the condition appear.
