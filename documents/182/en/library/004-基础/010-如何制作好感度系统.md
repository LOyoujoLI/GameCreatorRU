## Start

First let's analyze our requirements.

**Favorability** is displayed in numerical form, usually from 0 to 100, so here we can use **numerical variables**.

There are also various stages of Favorability, Strange (0~40), Familiar (41~60), Friend (61~80), Lover (81~100). So here we can use **String Variables** and **Conditional Branch** .

Different stages of good feelings will also trigger different storylines. So here we can use **conditional disagreement**.

## Creating Favorability values

You can control the value of the favorability variable by using **Numeric Variables**, and we can give different variables for different NPCs.

Since the default value of the numeric variable is 0, we don't need to set the default value of Favorability to 0, we can just use it.
